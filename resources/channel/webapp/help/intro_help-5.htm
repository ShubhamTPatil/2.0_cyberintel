If you are using a Windows platform, you can subscribe to the CM Documentation channel. You can download the contents to a Windows machine in the following ways:

    • Using the CM console
    • Using Channel Manager

Using the CM Console

To install the CM Documentation channel

  1. Log in to the CM console as a primary administrator.
  2. Choose Applications > Infrastructure > Tuner Administration.
  3. Select a tuner, and if necessary, enter the tuner admin user name (default is admin) and password (default is no password).
  4. Click Manage Channels and then click the Packages tab.
  5. Click Subscribe to a Package.
  6. In the Package URL field, enter the path http://product.marimba.com/Current/Version7/DocsChannel or click Browse and select the channel path on the BMC CM transmitter.
  7. Click Subscribe.

    8. The Documentation channel appears under My Channels with a status of Install-Pending.

  8. Under Actions, select Install.
  9. Click Refresh.
  10. When the Status field changes to Running, check your Windows task bar for an Application Installer icon.

  11. Click the Application Installer icon.
  12. Using the Setup wizard, install the documentation.
  13. Click Finish.

    14. The documents are now installed.

Using Channel Manager

To install the CM Documentation channel

  1. Using Channel Manager, select Browse.
  2. In the URL field, enter or select the following URL:

    3. http://product.marimba.com/Current/Version7

  3. From the list, select BMC CM Product Documentation.

    4. The documentation channel appears on the Channel Manager under My Channels.

  4. Right-click the channel and select Install from the menu.
  5. Using the Setup wizard, install the documentation.
  6. Click Finish.

    7. The documents are now installed.
