The BMC Configuration Management documentation is located on the BMC Customer Support website. On October 15, 2006, the Customer Support website changed to a new centralized BMC Customer Support website. Documentation published in October 2006 and earlier might refer to the previous website and email address.

Locating Configuration Management documentation

Configuration Management documentation is organized by product name on the website.

To locate a Configuration Management document

  1. Go to http://www.bmc.com/support_home. To access the Customer Support website, you must have a support contract.
  2. On the Support Central page, click Product Documentation.
  3. On the Product Documentation page, click Supported Product A-Z List.
  4. To locate CM documentation, use any of the following methods:
    • Enter the product name in the Search field, click Submit, and then click the product link.
    • From the product list, select the appropriate product name.

      • The following table lists which product name to use.

      For this CM document:
      Use this product name:
      Patch Management
      Patch Manager Pack for Clients
      Application Packager
      Application Manager for Clients
      CM Configuration Discovery Integration for CMDB
      Configuration Discovery for Servers
      All other CM documents
      Configuration Management Control Center

      The supported versions for Configuration Management are displayed.

  5. Select a CM version.
  6. Select a document.

Proactive notification

By subscribing to proactive notification, you can receive email messages that direct you to technical bulletins, release notes, and other documentation notices that are made available on the Customer Support site after you receive your initial product shipment. For more information about proactive notification, see the Customer Support page.
