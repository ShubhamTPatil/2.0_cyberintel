Wenn Sie Packages einer Policy hinzufügen, können Sie angeben, welche Packages von der Blackout-Periode ausgenommen werden sollen. Die Ausnahme ist dann nützlich, wenn Sie bei der Aktualisierung oder Installation von Packages während einer Blackout-Periode Notfallsituationen erwarten.

Bei diesem Vorgang wird davon ausgegangen, dass Sie sich auf der Seite "Policy bearbeiten" auf der Registerkarte "Packages" befinden. Weitere Informationen dazu finden Sie unter Allgemeine Anweisungen zum Erstellen und Bearbeiten von Policys.

So nehmen Sie Packages von der Blackout-Periode aus:

  1. Suchen Sie auf der Registerkarte "Packages" der Seite "Policy bearbeiten" die Zeilen der Packages, die von der Blackout-Periode ausgenommen werden sollen.
  2. Aktivieren Sie in den Zeilen der entsprechenden Packages die Kontrollkästchen unter der Spalte Vom Blackout ausgenommen.

Damit diese Ausnahmen wirksam werden, müssen Sie auch Policy Service von der Blackout-Periode ausnehmen, so dass dieser Policy-Aktualisierungen abrufen und anwenden kann. Die Option zur Ausnahme von Policy Service wird beim Einrichten der Blackout-Periode festgelegt. Weitere Informationen dazu finden Sie unter Festlegen der Blackout-Periode für ein Ziel.

Nachdem Sie die Policy bearbeitet haben, sehen Sie sich die Vorschau an, und speichern Sie die Policy. Weitere Informationen dazu finden Sie unter Erstellen der Vorschau und Speichern von Policy-Änderungen.
