Entscheiden Sie am besten, ob die ACL-Funktion während der Installation und während des Setups verwendet oder nicht verwendet werden soll. Um zu sehen, ob die ACL-Funkion aktiviert ist oder nicht, gehen Sie mit dem Mauszeiger auf das Statussymbol rechts oben im Konsolenfenster.

Hinweis: Es wird empfohlen, den Administratoren ACLs und Berechtigungen zuzuweisen, bevor die ACL-Funktion aktiviert wird. Andernfalls können Administratoren, die sich bei Policy Manager anmelden, keine Policys verwalten. Am besten ist es, den Administratoren ACLs und Berechtigungen zuzuweisen, diese "bereitzustellen" und dann die ACL-Funktion in Policy Manager zu aktivieren, wenn Berechtigungen angewendet werden sollen.

Bevor Sie mit Policy Manager die Zugriffskontrollfunktion aktivieren, richten Sie mit der Konsole Folgendes ein:

    • Benutzerauthentifizierung
    • LDAP-Synchronisierung
    • Zugriffskontrolllisten
    • Berechtigungen für Zugriffskontrolllisten zuweisen
    • Berechtigungen für Applikationen zuweisen

Weitere Informationen dazu finden Sie in der Konsolenhilfe für die Systemeinstellungen.

Aktivieren und Deaktivieren der ACL-Funktion

Nach der Einrichtung von Zugriffskontrolllisten können Primäradministratoren mit Policy Manager die ACL-Funktion aktivieren. ACLs sind standardmäßig deaktiviert.

Die ACL-Funktion kann über die Registerkarte "Konfiguration" oder die Befehlszeilenschnittstelle aktiviert oder deaktiviert werden.

So aktivieren Sie die ACL-Funktion mit Hilfe der Konfigurationsoptionen:

    • Klicken Sie auf die Registerkarte "Konfiguration".
    • Klicken Sie auf "ACL-Optionen".

      • Die Seite "Zugriffskontrolle aktivieren" wird angezeigt.

    • Wählen Sie das Kontrollkästchen "Zugriffskontrolle aktivieren/deaktivieren" aus.
    • Klicken Sie auf OK.

So aktivieren Sie die ACL-Funktion mit Hilfe der Befehlszeile:

    • Geben Sie in der Befehlszeile den folgenden Befehl ein:

      • runchannel <Policy_Manager_URL> -user <Benutzername> -password <Passwort> -configSet marimba.subscription.acl true

      Sie können die ACL-Funktion mit demselben Befehl deaktivieren, indem Sie true durch false ersetzen. Außerdem können Sie mit der Option -preview die Änderungen, die Sie vornehmen möchten, in der Vorschau anzeigen.

      Achten Sie darauf, dass Sie Policy Manager neu starten, nachdem die ACL-Funktion aktiviert wurde.
