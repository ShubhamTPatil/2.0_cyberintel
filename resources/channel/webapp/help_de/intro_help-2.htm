Mit Policy Manager können Sie als Administrator Packages Zielen zuweisen. Diese Ziele stehen für einen oder mehrere Endpunkte wie Benutzer, Computer oder Gruppen. Für jedes Ziel kann dem Package ein Status zugewiesen werden wie "bereitstellen", "installieren" oder "deinstallieren". Der angegebene Status wird auf das Package bei jedem Ziel nach einem von Ihnen festgelegten Zeitplan angewendet. Policy Manager ist Browser-basiert und speichert Informationen zu Packages und Zielen in einem Verzeichnisdienst unter Verwendung des LDAP-Protokolls.

Dieses Dokument richtet sich an Netzwerkmanager, Systemadministratoren, Help-Desk-Mitarbeiter und IT-Manager, die Policy Manager bereitstellen und konfigurieren oder zur Verwaltung eines Computernetzwerks benutzen. Das Dokument basiert auf der Annahme, dass Sie mit BMC Configuration Management (CM)-Produkten, wie Tunern und Transmittern, vertraut sind.

Dieses Produkt umfasst die folgenden Komponenten:

    • Policy Manager-Channel. Dieser enthält eine Browser-basierte Schnittstelle, die verwendet wird, um Policys für die Verteilung von Packages an Ziele zu erstellen, um Packages einen Status zuzuweisen und um festzulegen, wann Packages heruntergeladen und installiert werden. Sie können mit dieser Komponente auch Konfigurationseinstellungen angeben. Policy Manager verfügt daneben über eine Befehlszeilenschnittstelle zur Konfiguration einiger Einstellungen, die für die Verteilung von Packages an Ziele verwendet werden.
    • Common Management Services-Channel. Dieser ermöglicht die Konfiguration von systemweiten Einstellungen, die für Policy Manager und andere vorhandene BMC CM-Web-Applikationen gelten.
    • Policy Service-Channel. Dabei handelt es sich um eine Client-seitige Komponente. Sie ist für die Anwendung der Policys zuständig, die dem Ziel-Tuner, auf dem sie ausgeführt wird, zugewiesen werden.

Zusätzlich benötigen Sie einen der folgenden Verzeichnisdienste:

    • Microsoft Active Directory (AD)
    • Microsoft Active Directory-Applikationsmodus (ADAM)
    • Sun ONE Directory

Informationen dazu, welche Versionen dieser Verzeichnisdienste unterstützt werden, finden Sie in den Veröffentlichungshinweisen für Policy Management, die auf der BMC Customer Support-Website verfügbar sind.
