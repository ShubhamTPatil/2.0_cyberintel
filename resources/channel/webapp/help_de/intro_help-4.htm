Der Zugriff auf die BMC Configuration Management (CM)-Dokumentation hat sich verändert.

Die BMC CM-Dokumentation befindet sich jetzt auf der BMC Customer Support-Website. Dafür benötigen Sie einen Anmeldenamen und ein Passwort. Sie können Dokumente nach Produktname, Version, Sprache oder Dokumenttyp suchen. Die Suche kann bei Bedarf eingeengt oder erweitert werden, wenn Sie beispielsweise nur nach Dokumenten für die Version 6.0.3 oder nur nach Policy Management Version 6.0.3 suchen möchten. Je weniger Kriterien bei der Suche verwendet werden, desto mehr Suchergebnisse erhalten Sie.

So suchen Sie nach einem Dokument:

  1. Gehen Sie zu http://supportweb.remedy.com/bmc_cm.
  2. Geben Sie im Feld "My Support Login" Ihren Benutzernamen und Ihr Passwort ein, und klicken Sie dann auf Login.
  3. Klicken Sie auf Documentation.
  4. Wählen Sie eines der folgenden Suchkriterien aus.
  5. Ein Produkt, z. B. Policy Management.
  6. Eine Version, z. B. 6.0.3.
    • Wenn Sie ein Produkt auswählen, können Sie aus der Liste eine Version auswählen.
    • Wenn Sie kein Produkt auswählen, können Sie eine Versionsnummer in das Versionsfeld eingeben.
    • Wenn Sie die Vorgänger- und die Nachfolgerversion anzeigen möchten, nutzen Sie dazu die Symbole ">" (größer als) und "<" (kleiner als). Beispiel: Sie geben ">6" ein, um die Versionen 6.0, 6.0.1, 6.0.2, 6.0.2.1, 6.0.3 und 6.5 anzuzeigen.
    • Eine Sprache, z. B. Englisch oder Deutsch.
    • Einen Dokumententyp, z. B. Handbuch.

      • Hinweis: Eine vollständige Liste der BMC CM-Dokumente können Sie anzeigen lassen, indem Sie keines der Suchkriterien auswählen.

  7. Klicken Sie auf "Search Documentation".

    8. Die Suchergebnisse werden angezeigt.

    • Sie können bei den Suchergebnissen auf den Link in der Spaltenüberschrift klicken und die Dokumente nach dieser Spaltenüberschrift sortieren.
    • Wenn die Suchergebnisse eine große Anzahl an Dokumenten umfassen, können Sie über der Ergebnisliste auf Next oder Previous klicken, um weitere Dokumente anzuzeigen.
  8. Klicken Sie auf ein Dokument in den Suchergebnissen, um es auszuwählen.

    9. Im Feld Documentation Notes wird eine Beschreibung des Dokuments angezeigt.

  9. Führen Sie einen der folgenden Schritte aus:
    • Klicken Sie auf View, um das ausgewählte Dokument anzuzeigen.
    • Sie können jeweils nur ein Dokument anzeigen. Das geöffnete Dokument muss geschlossen werden, bevor ein anderes Dokument angezeigt werden kann.
    • Klicken Sie auf Save To Disk, um das ausgewählte Dokument auf der Festplatte zu speichern.

      • Klicken Sie auf New Search, um eine neue Suche durchzuführen.

Die Online-Hilfe zu Policy Manager ist dafür vorgesehen, zusammen mit dem Administratorhandbuch für Policy Management verwendet zu werden, das ebenfalls auf der BMC Customer Support-Website zu finden ist. Das Administratorhandbuch für Policy Management enthält Informationen zur Unterstützung von Plattformen, zu Installation und Aktualisierungen, zur Einstellung des Verzeichnisdiensts (LDAP), zur Wartung und zur Fehlerbehebung. Das Drucken des Handbuchs wird durch ein Inhaltsverzeichnis, einen Index und Seitenzahlen erleichtert.

Folgende zugehörige Dokumente können Sie ebenfalls auf der BMC Customer Support-Website finden:

    • Im Handbuch über die Systemvoraussetzungen für das BMC Configuration Management (BMC Configuration Management System Requirements Guide) sind die Hardwarevoraussetzungen (z. B. Verarbeitungsgeschwindigkeit, Speicherplatz und RAM) sowie die Anforderungen an das Betriebssystem für die verschiedenen unterstützten Plattformen aufgelistet. In diesem Dokument sind auch die unterstützten Datenbanken und Verzeichnisdienste aufgeführt.
    • Das BMC CM-Planungshandbuch (BMC Configuration Management Planning Guide) hilft Ihnen beim Entwerfen einer BMC CM-Infrastruktur für Ihr Unternehmen.
    • Das BMC CM Installations- und Deployment-Handbuch (BMC Configuration Management Installation and Deployment Guide) bietet Anweisungen für eine Neuinstallation (Deployment) Ihrer BMC CM-Infrastruktur.
    • Das Administratorhandbuch für Report Center (Report Center Administrator’s Guide) enthält genaue Anweisungen zur Installation aller Komponenten des Inventory Management-Moduls. Es zeigt außerdem, wie Sie Inventory-Informationen und zentralisierte Logging-Informationen abfragen und wie Sie die Inventory- und Logging-Plugins konfigurieren.
    • Das Administratorhandbuch für Server Management (Server Management Administrator's Guide)ist die Hauptdokumentation für die Server Management-Produkte. Es bietet konzeptionelle Informationen über die Verwendung von Server Management, inklusive der Browser-basierten Schnittstelle.
    • Das Administratorhandbuch für Application Packager (Application Packager Administrator's Guide) beschreibt ausführlich, wie Sie mit Application Packager Software-Packages zur Verteilung auf Desktops oder Servern erstellen. Außerdem sind Informationen über die Verwendung der Befehlszeile, Policys, XML-Vorlagen und Windows-System-Makros enthalten.
    • Das BMC CM-Referenzhandbuch (Configuration Management Reference Guide) bietet eine große Auswahl an Referenzinformationen über die Komponenten der BMC CM-Infrastruktur, wie zum Beispiel Befehlszeilenoptionen, Tuner-Eigenschaften, Channel-Eigenschaften, Channel-Parameter, Channel-Status und Logging-Codes. Es listet die Tuner- und Package-Eigenschaften auf, die Sie mit Policy Manager festlegen können. Außerdem werden Befehlszeilenoptionen und Log-Einträge aufgelistet, die bei der Verwendung von Policy Manager hilfreich sind.
    • Der BMC CM Aktualisierungshandbuch (BMC Configuration Management Upgrade Guide( zeigt, wie Sie Ihre BMC CM-Infrastruktur auf die aktuellste Version aktualisieren.
    • Weitere Dokumenten, die Sie möglicherweise unter bestimmten Umständen verwenden möchten, werden im entsprechenden Abschnitten dieses Dokuments erwähnt.
