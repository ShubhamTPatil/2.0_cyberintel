Informationen zur Installation der in diesem Abschnitt erwähnten Komponenten finden Sie im BMC CM Installations- und Deployment-Handbuch (BMC Configuration Management Installation and Deployment Guide), das auf der BMC Customer Support-Website verfügbar ist. Informationen zur Konfiguration von Patch Manager und Patch Service finden Sie im Administratorhandbuch für Patch Management. Informationen zur Konfiguration von Report Center und Scanner Service finden Sie im Administratorhandbuch für Report Center. Beide Dokumente sind auch auf der BMC CM-Seite "Customer Support" erhältlich.

Serverseitig. Folgende Komponenten müssen auf demselben Computer wie Policy Manager installiert und konfiguriert sein:

    • Patch Manager
    • Report Center

Clientseitig. Folgende Komponenten müssen auf den gleichen Clients wie Policy Service installiert und konfiguriert sein:

    • Patch Service
    • Scanner Service
