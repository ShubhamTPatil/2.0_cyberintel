Der Patch-Gruppe-Zuweisungsstatus bestimmt, ob eine Patch-Gruppe Zielcomputern zugewiesen werden soll oder nicht. Es gibt zwei Zuweisungsstatus, die Sie Patch-Gruppen in Policy Manager zuweisen können:

    • Zuweisen - Wählen Sie diesen Status, wenn Sie eine Patch-Gruppe den angegebenen Zielcomputern zuweisen möchten.
    • Ausschließen - Wählen Sie diesen Status, wenn Sie eine Patch-Gruppe den angegebenen Zielcomputern nicht zuweisen möchten. Wenn Sie beispielsweise eine Patch-Gruppe einer Gruppe von Computern mit Ausnahme eines Computers zuweisen möchten, wählen Sie für die gesamte Gruppe zuweisen aus. Für den Computer, der ausgenommen werden soll, wählen Sie ausschließen aus.

Der Patch-Gruppe-Zuweisungsstatus ist nicht mit der Patch-Aktion zu verwechseln. Die Patch-Aktion bestimmt, was mit jedem Patch einer Patch-Gruppe auf einem Zielcomputer geschehen soll, wie etwa „installieren“, „bereitstellen“, „deinstallieren“ oder nichts. Die Aktion für die einzelnen Patches einer Patch-Gruppe wird mit Hilfe von Patch Manager angegeben.
