Wenn Sie eine Patch-Gruppe aus einer Policy löschen, wird die Patch-Gruppe bei der nächsten Aktualisierung durch Policy Service auf dem Ziel gelöscht. Das Entfernen der Patch-Gruppe führt jedoch nicht dazu, dass Patches in der Patch-Gruppe auf dem Ziel deinstalliert werden.

Um Patches zu deinstallieren, müssen Sie mit Patch Manager folgende Schritte durchführen:

  1. Bearbeiten Sie die Patch-Gruppe.
  2. Legen Sie die Deinstallationsaktion für einzelne Patches fest, die Sie deinstallieren möchten.
  3. Veröffentlichen Sie die Patch-Gruppe neu.

Bei der nächsten Ausführung von Patch Service wird die Patch-Gruppe aktualisiert und die angegebenen Patches werden deinstalliert. Weitere Informationen finden Sie im Administratorhandbuch für Patch Management, das auf der BMC CM-Seite "Costumer Support" erhältlich ist.
