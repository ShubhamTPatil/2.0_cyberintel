Sie können die Simulation einer Patch-Installation nur dann anzeigen, wenn die folgenden Service-Channel mindestens einmal auf dem Zielclient ausgeführt wurden.

    • Patch Service
    • Scanner Service

Wenn Sie ausgeführt werden, sammeln diese Service-Channel Informationen über die simulierte Installation von Patches.

Beim Simulieren einer Patch-Installation wird die Simulation für alle Patch-Gruppen, die zum ausgewählten Ziel gehören, durchgeführt, nicht nur für die in der Policy enthaltenen. Die Simulation schließt auch die Patch-Gruppen ein, die allen Clients (sowohl Container als auch Gruppen), zu der das ausgewählte Ziel gehört, zugewiesen wurden.

Hinweis: Zur Simulation einer Patch-Installation sollte das für Patch-Gruppen ausgewählte Ziel ein einzelner Computer sein.

So simulieren Sie die Installation von Patch-Gruppen

Klicken Sie auf der Registerkarte „Patch-Gruppen“ auf Simulation ausführen.

Die Seite „Patch-Installation simulieren“ wird angezeigt. Auf ihr wird die Reihenfolge für die Installation der Patches auf dem Zielcomputer angezeigt. Zusätzlich erhalten Sie auf der Seite folgende Informationen:

    • Eine kurze Beschreibung des Patches.
    • Die ID, die dem Patch von Patch Manager zugewiesen wurde. Diese ID unterscheidet sich von der ID, die der Anbieter des Patches diesem zugewiesen hat.
    • Die Aktion, die Sie mit Hilfe von Patch Manager für den Patch angegeben haben.
    • Hinweise, die der Anbieter mit dem Patch zur Verfügung gestellt hat.

      • Sie erhalten weitere Informationen zu diesen Patches, wenn Sie auf Patch-Simulations-Logs anzeigen klicken. Weitere Informationen dazu finden Sie unter Anzeigen weiterer Informationen zur Installation von Patch-Gruppen.
