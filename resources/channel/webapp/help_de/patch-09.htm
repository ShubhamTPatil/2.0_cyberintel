So zeigen Sie weitere Informationen zur Installation von Patch-Gruppen an

Klicken Sie auf der Seite „Patch-Installation simulieren“ auf Ausführliche Logs anzeigen.

Die Seite „Patch-Simulations-Logs“ wird angezeigt. Auf ihr werden weitere Informationen über die Patches aufgeführt. So können Sie erkennen, welche Patches veraltet sind oder welche Patches aufgrund von Konflikten nicht installiert werden können.

Filtering patch groups

checking MS04-004.Q832894.Q832894-6SP1.exe._1139234174 for obsolescence

checking MS04-007.Q828028.Windows2000-KB828028-x86-ENU.exe._1592190521 for obsolescence
