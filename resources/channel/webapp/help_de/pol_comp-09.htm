Finden von Zielen und Durchführen von Abfragen

So finden Sie ein Ziel und führen eine Compliance-Abfrage durch:

  1. Suchen Sie ein Ziel auf der Registerkarte Zielansicht.

    2. Sie haben auch die Möglichkeit, mit der Registerkarte Ziel-Compliance zu beginnen.

    • Suchen Sie im linken Bereich das Ziel, für das Compliance-Informationen angezeigt werden sollen, und wählen Sie dieses aus.

  2. Klicken Sie auf Compliance, um die Compliance-Informationen anzuzeigen.

    3. Der Zielname wird auf der Registerkarte Ziel-Compliance oben im rechten Bereich angezeigt. Wenn Ihr Ziel Mitglieder enthält, können die Policy-Compliance oder die Gesamt-Compliance angezeigt werden. Die Registerkarte Policy Compliance ist standardmäßig aktiviert.

Anzeigen der Policy Compliance für Ziele

Es können Compliance-Informationen für Ziele angezeigt werden, die Mitglieder enthalten, oder Ziele, die einzelne Computer sind. Die folgenden Verfahren gelten für alle Ziele.

So zeigen Sie die Policy Compliance für ein Ziel an:

  1. So zeigen Sie Compliance-Informationen für alle zugewiesenen Packages im ausgewählten Ziel an:
    • Suchen Sie ein Ziel auf der Registerkarte Ziel-Compliance.
    • Suchen Sie im linken Bereich das Ziel, für das Compliance-Informationen angezeigt werden sollen, und wählen Sie dieses aus. Der Zielname wird oben im rechten Bereich angezeigt, und die Registerkarte Policy Compliance ist aktiv.
    • Wählen Sie alle Packages aus, indem Sie in der Kopfzeile das Kontrollkästchen aktivieren. Klicken Sie dann auf "Berechnen".

  2. So zeigen Sie Compliance-Informationen für bestimmte Packages an, die Sie einem ausgewählten Ziel zugewiesen haben:
    1. Aktivieren Sie in der Registerkarte Policy Compliance das Kontrollkästchen aller Packages, die in die Abfrage eingeschlossen werden sollen.
    2. Klicken Sie auf Berechnen.

      3. Policy Manager aktualisiert die ausgewählten Packages mit den Statusinformationen zu Ihrer Abfrage. In der folgenden Tabelle werden die verfügbaren Status angezeigt.

      Abfragestatus
      Beschreibung
      Nicht berechnet
      Keine Compliance-Informationen verfügbar.
      Wird berechnet
      Policy Manager hat Ihre Abfrage noch nicht der Compliance-Warteschlange hinzugefügt.
      In Warteschlange
      Die Abfrage ist in der Compliance-Warteschlange.
      Wird verarbeitet
      Die Abfrage wird auf dem Compliance-Server ausgeführt.
      Fehler
      Die Abfrage konnte nicht ausgeführt werden.

  3. So zeigen Sie Informationen zu Clients an, die konform, nicht konform oder nicht eingecheckt sind:
    1. Klicken Sie auf das Rautenzeichen (#) im Compliance-Spaltenkopf, um die genaue Anzahl der Clients für jeden Status anzuzeigen.
    2. Klicken Sie auf das Prozentzeichen (%) im Compliance-Spaltenkopf, um die Anzahl der Clients als Prozentsatz der Gesamtanzahl von Clients im Ziel anzuzeigen.

      3. Die Anzahl oder der Prozentsatz wird in unterschiedlichen Farben angezeigt, die den Status angeben:

    3. Grün für konform.
    4. Rot für nicht konform.
    5. Blau für nicht eingecheckt.

      6. Klicken Sie auf die angezeigte Anzahl oder den angezeigten Prozentsatz, um eine Liste der Computer aufzurufen, welche mit Packages mit einem bestimmten Status verknüpft sind.

  4. So zeigen Sie Compliance-Informationen für alle Computer im ausgewählten Ziel an:
    1. Klicken Sie auf die Registerkarte Gesamt-Compliance. Die Compliance-Status-Registerkarten in grün, rot und blau stehen für die Computer in dem ausgewählten Ziel, die konform, nicht konform bzw. nicht eingecheckt sind.
    2. Klicken Sie auf die grüne, rote oder blaue Compliance-Status-Registerkarte, um die Liste der Computer für jeden Status anzuzeigen.

      3. Bei der Berechnung des Compliance-Status berücksichtigt Policy Manager nur die Computer, die während des Zeitraums eingecheckt haben, den Sie auf der Registerkarte "Konfiguration" bei den Compliance-Optionen festgelegt haben. Andernfalls betrachtet Policy Manager die Computer als nicht eingecheckt. Weitere Informationen finden Sie unter Konfigurieren der Policy Compliance-Einstellungen.

    3. Auf der grünen Registerkarte werden alle Computer in diesem Ziel angezeigt, die mit sämtlichen (sowohl direkt als auch indirekt) zugewiesenen Policys konform sind.
    4. Auf der roten Registerkarte werden alle Computer in diesem Ziel angezeigt, die mit sämtlichen (sowohl direkt als auch indirekt) zugewiesenen Policys nicht konform sind. In der Liste wird für jeden Computer ein Grund für die Nicht-Compliance angezeigt.

      5. Hinweis: In der Liste wird aus Performance-Gründen nur ein Grund für die Nicht-Compliance angezeigt. Klicken Sie auf den Computernamen, um weitere Informationen dazu anzuzeigen, weshalb der Computer nicht konform ist.

    5. Auf der blauen Registerkarte werden alle Computer in diesem Ziel angezeigt, die sich nicht eingecheckt haben, seitdem Änderungen an einer Policy vorgenommen wurden, die diese Computer betreffen. Für jeden Computer wird in der Liste das Datum angegeben, an dem der Computer zuletzt einen Status gemeldet hat.

      6. Überprüfen der Abfrageinformationen

      Um die Einrichtung und die Ergebnisse der Abfrage in Report Center zu überprüfen, klicken Sie auf die Schaltfläche Report Center-Abfrage anzeigen.

      Generieren eines Berichts

Um einen Gesamt-Compliance-Bericht für ein Ziel zu erstellen, klicken Sie auf die Schaltfläche Compliance-Bericht generieren. Bestätigen Sie die Meldung im Fenster, und die Registerkarte Compliance-Berichte wird angezeigt.

Hinweis: Berichte benötigen einige Zeit für die Ausführung. Wenn Sie auf die Schaltfläche Compliance-Bericht generieren klicken, wird sofort nach der Bestätigungsmeldung die Startzeit angezeigt. Die Endzeit wird erst angezeigt, wenn die Ausführung des Berichts abgeschlossen ist.

Weitere Informationen dazu finden Sie unter Compliance-Berichte.

Anzeigen der Policy Compliance für Packages

Sie können die Policy Compliance-Informationen für ein Package anzeigen, um Folgendes festzustellen:

    • Welche Ziele sollten über dieses Package verfügen?
    • Verfügen diese Ziele über dieses Package?

Das Anzeigen der Package-Compliance ist mit dem Anzeigen der Ziel-Compliance vergleichbar. Statt der Registerkarte Ziel-Compliance zur Abfrage von mit einem Ziel verknüpften Packages verwenden Sie die Registerkarte Package-Compliance, um mit einem Package verknüpfte Ziele abzufragen.

Mit dem Suchbereich können Sie die Liste der Packages finden und filtern, die Sie in die Abfrage einschließen möchten. Suchen Sie dabei nach dem Package-Namen oder dem Veröffentlichungsdatum. Ein Sternchen (*) und das Prozentzeichen (%) werden von der Suchfunktion als Platzhalter unterstützt. Siehe So zeigen Sie die Policy Compliance für ein Ziel an:.
