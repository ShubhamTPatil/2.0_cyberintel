Zusätzlich zu den schnell ausgeführten Compliance-Abfragen zu Zielen oder Packages können Sie Berichte ausführen, die die Gesamt-Ziel-Compliance berechnen und zur späteren Überprüfung speichern. Sie starten Berichte auf der Seite Ziel-Compliance auf der Registerkarte Gesamt-Compliance, indem Sie auf die Schaltfläche Compliance-Bericht generieren klicken. Die Berichte werden dann für ein späteres Überprüfen oder Entfernen auf der Registerkarte Compliance-Berichte gespeichert. (Weitere Informationen zum Ausführen von Ziel-Compliance-Abfragen finden Sie unter So zeigen Sie die Policy Compliance für ein Ziel an:.)

Über die Registerkarte Compliance-Berichte können Sie gespeicherte Compliance-Berichte auch löschen.

So zeigen Sie Compliance-Berichte an oder löschen diese:

  1. Klicken Sie in Policy Manager auf die Registerkarte Compliance-Berichte.

    2. Die gespeicherten Compliance-Berichte werden angezeigt. Sie können für jeden Bericht zusätzliche Informationen anzeigen:

    • Halten Sie den Mauszeiger auf die Prozentzahlen für "Konform", "Nicht konform" bzw. "Nicht eingecheckt", um die Anzahl der Computer mit dem jeweiligen Status anzuzeigen.
    • Klicken Sie auf die Prozentzahlen für "Konform", "Nicht konform" oder "Nicht eingecheckt", um eine Liste der Computer mit dem jeweiligen Status anzuzeigen.
  2. Um Berichte zu löschen, wählen Sie die zu entfernenden Berichte aus und klicken auf Löschen.
