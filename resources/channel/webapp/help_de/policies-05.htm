Policy Manager verlangt, dass Sie eine Vorschau der Policy-Änderungen anzeigen, bevor Sie diese abspeichern können. Dadurch soll sichergestellt werden, dass vom Ziel nur korrekte Einstellungen übernommen werden. Beachten Sie, dass das Speichern einer Policy dazu führt, dass die Änderungen, die Sie auf allen Registerkarten vorgenommen haben, gespeichert werden. Überprüfen Sie die Änderungen auf allen Registerkarten, bevor Sie sie übernehmen.

Die Seite "Vorschau" entspricht der Seite "Policy bearbeiten". Die Seite "Vorschau" umfasst die folgenden vier Registerkarten. Dabei entspricht jede Registerkarte einer Registerkarte auf der Seite "Einstellungen bearbeiten":

    • Packages: Zeigt die Packages der Policy an. Dies umfasst auch Informationen über jedes Package, wie Status, Zeitpläne und Installationspriorität.
    • Blackout-Periode: Zeigt die Blackout-Periode des Ziels an.
    • Policy-Zeitplan: Zeigt den Aktualisierungszeitplan Policy Service an.
    • Erweitert: Zeigt erweiterte Einstellungen an, wie Tuner- und Package-Eigenschaften, Transmitter-Berechtigungen und Profilinformationen für das Ziel.

So können Sie eine Vorschau anzeigen und die Änderungen speichern, verwerfen oder weiter verändern:

  1. Klicken Sie nach dem Bearbeiten der Policy auf Vorschau.

    2. Wenn Sie auf der Seite "Policy bearbeiten" auf "Vorschau" klicken, wird die Seite "Vorschau" angezeigt. Klicken Sie auf jede Registerkarte, um eine Vorschau der Änderungen anzuzeigen, die Sie dort vorgenommen haben.

  2. Wenn Sie die Policy speichern möchten, klicken Sie auf Speichern. Dadurch werden die Änderungen bestätigt und gespeichert.
  3. Wenn Sie die Änderungen nicht speichern, sondern die Policy weiter bearbeiten möchten, klicken Sie auf Zurück.
  4. Sie können Ihre Änderungen jederzeit verwerfen und zur Seite "Zielansicht" zurückkehren, indem Sie auf Abbrechen klicken.
