Das Benutzerkonto (im Verzeichnisdienst), das Sie verwendet haben, um sich bei der Konsole und bei Policy Manager anzumelden, wird auch zum Speichern der von Ihnen erstellten bzw. bearbeiteten Policys verwendet. Sie können Policys nicht speichern, wenn Sie für den untergeordneten Container im Verzeichnisdienst keine Schreibberechtigung besitzen. Der untergeordnete Container wurde erstellt, als Sie (oder der Primäradministrator) Policy Manager eingerichetet haben. Weitere Informationen dazu finden Sie im Abschnitt über Installationsskripte im Administratorhandbuch für Policy Management, das auf der BMC Customer Support-Website verfügbar ist.

Wenn Sie Active Directory verwenden und die gespeicherte Policy für ein Ziel bestimmt ist, das sich an einem anderen Standort als die Policys befindet, wird die Policy wirksam, wenn die Policys auf den dem Plugin am nächsten gelegenen globalen Katalog repliziert werden.
