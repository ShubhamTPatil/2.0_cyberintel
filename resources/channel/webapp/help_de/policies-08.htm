Bei diesem Vorgang wird davon ausgegangen, dass Sie sich auf der Seite "Package-Liste bearbeiten" befinden. Zu dieser Seite gelangen Sie, indem Sie auf der Seite "Policys bearbeiten" auf der Registerkarte "Packages" auf "Package-Liste bearbeiten" klicken. Weitere Informationen dazu finden Sie unter Allgemeine Anweisungen zum Erstellen und Bearbeiten von Policys.

So fügen Sie Packages zu einer Policy hinzu:

  1. Wählen Sie links auf der Seite "Package-Liste bearbeiten" die Packages aus, die Sie der Policy hinzufügen möchten.
    • Klicken Sie auf die Registerkarte Aktuell bereitgestellt, wenn das Package, das Sie der Policy hinzufügen möchten, bereits einem Ziel zugeordnet ist.
    • Klicken Sie auf die Registerkarte Transmitter, wenn Sie ein Package auswählen möchten, das bereits auf einem Transmitter veröffentlicht, aber noch keinem Ziel zugewiesen wurde. Sie können die Liste der Packages auf einem Transmitter durchsuchen, indem Sie die URL-Adresse des Transmitter eingeben und auf Start klicken.

      • Jedes Mal, wenn Sie ein Package auswählen, wird dieses der Package-Liste rechts auf der Seite hinzugefügt. Wenn Sie einen Fehler gemacht haben, wählen Sie das Package aus, das Sie aus der Package-Liste entfernen möchten, und klicken Sie auf Entfernen.

  2. Klicken Sie auf OK, um die Änderungen zu speichern und zur Seite "Policy bearbeiten" zurückzukehren.

Nachdem Sie die Policy bearbeitet haben, sehen Sie sich die Vorschau an, und speichern Sie die Policy. Weitere Informationen dazu finden Sie unter Erstellen der Vorschau und Speichern von Policy-Änderungen.
