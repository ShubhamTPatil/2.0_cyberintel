Bei diesem Vorgang wird davon ausgegangen, dass Sie sich auf der Seite "Package-Liste bearbeiten" befinden. Zu dieser Seite gelangen Sie, indem Sie auf der Seite "Policys bearbeiten" auf der Registerkarte "Packages" auf "Package-Liste bearbeiten" klicken. Weitere Informationen dazu finden Sie unter Allgemeine Anweisungen zum Erstellen und Bearbeiten von Policys.

Wenn Sie ein Package aus einer Policy löschen, wird das Package auf dem Ziel entfernt, sobald Policy Service die nächste Aktualisierung durchführt. Das Package wird allerdings in folgenden Fällen nicht auf dem Ziel gelöscht:

    • Der Package-Status ist auf Installieren-Beibehalten oder auf Installieren-Starten-Beibehalten gesetzt.
    • Die Tuner-Eigenschaft Nicht Löschen (marimba.subscription.nodelete) ist auf true gesetzt.

In den oben genannten Fällen bleibt das Package auf dem Ziel bestehen, es wird allerdings nicht mehr mit Policy Manager verwaltet. Um das Package auf den Zielen wirklich zu löschen, müssen Sie den Primärstatus des Package ausdrücklich auf Deinstallieren setzen.

So löschen Sie Packages aus einer Policy:

  1. Wählen Sie rechts auf der Seite "Package-Liste bearbeiten" die Packages aus, die Sie aus der Package-Liste entfernen möchten.
  2. Klicken Sie auf Entfernen.
  3. Klicken Sie auf OK, um die Änderungen zu speichern und zur Seite "Policy bearbeiten" zurückzukehren.

Nachdem Sie die Policy bearbeitet haben, sehen Sie sich die Vorschau an, und speichern Sie die Policy. Weitere Informationen dazu finden Sie unter Erstellen der Vorschau und Speichern von Policy-Änderungen.
