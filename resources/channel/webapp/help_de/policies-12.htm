Auf der Seite "Policy bearbeiten" können Sie für jedes Package der Policy einen Primärstatus festlegen. Eine Erläuterung der verschiedenen Package-Status finden Sie unter Übersicht über Installationsstatus.

Mit den beiden Primärstatus Bereitstellen und Ankündigen können Sie einen Sekundärstatus festlegen, mit dem die Installation des Package gesteuert werden kann. Sie können mit dem Sekundärstatus einen Aktivierungszeitplan verknüpfen, um für die Zukunft einen Zeitraum zu bestimmen, innerhalb dessen Package-Installationen im gesamten Unternehmen durchgeführt werden. Dies gilt selbst für Clients, die nur selten eine Verbindung zum Netzwerk herstellen. Der Standardwert des Sekundärstatus ist Installieren.

Weitere Informationen zu Primär- und Sekundärstatus finden Sie unter Übersicht über Installationsstatus.

Bei dem folgenden Vorgang wird davon ausgegangen, dass Sie sich auf der Seite "Policy bearbeiten" auf der Registerkarte "Packages" befinden. Weitere Informationen dazu finden Sie unter Allgemeine Anweisungen zum Erstellen und Bearbeiten von Policys.

So legen Sie Primär- und Sekundärstatus für ein Package fest:

  1. Suchen Sie auf der Seite "Policy bearbeiten" auf der Registerkarte "Packages" die Zeile mit dem Package, für das Primär- und Sekundärstatus festgelegt werden sollen.
  2. Wählen Sie in der Zeile des Package in der Spalte Primärstatus aus der Dropdown-Liste einen Status aus.
  3. Wenn ein Sekundärstatus festgelegt werden soll, wählen Sie in der Spalte Sekundärstatus aus der Dropdown-Liste einen Status aus. Der Standardwert des Sekundärstatus ist Installieren.
