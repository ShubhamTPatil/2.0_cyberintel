Sie können allgemeine Zeitpläne für mehrere Packages so festlegen, dass zwei oder mehr Packages in einer Policy die gleichen Zeitpläne besitzen. Wenn Sie allgemeine Zeitpläne für mehrere Packages festlegen, legen Sie alle Zeitpläne fest: Primär-, Sekundär-, Aktualisierungs- und Reparaturzeitpläne. Selbst wenn Sie nur einen Zeitplantyp bearbeiten, werden die anderen Zeitpläne, die auf der Seite "Allgemeinen Zeitplan festlegen" angezeigt werden, von allen Packages übernommen.

Bei diesem Vorgang wird davon ausgegangen, dass Sie sich auf der Seite "Policy bearbeiten" auf der Registerkarte "Packages" befinden. Weitere Informationen dazu finden Sie unter Allgemeine Anweisungen zum Erstellen und Bearbeiten von Policys.

So legen Sie allgemeine Zeitpläne für mehrere Packages fest:

  1. Aktivieren Sie auf der Seite "Policy bearbeiten" auf der Registerkarte "Packages" die Kontrollkästchen, die mit den Packages übereinstimmen, für die allgemeine Zeitpläne festgelegt werden sollen.
  2. Klicken Sie auf Allgemeinen Zeitplan festlegen.
  3. Legen Sie auf der Seite "Allgemeinen Zeitplan festlegen" die Primär-, Sekundär-, Aktualisierungs- und Reparaturzeitpläne für die Packages fest.

    4. So legen Sie beispielsweise den Zeitplan für den Primärstatus fest:

    1. Klicken Sie im Abschnitt Zeitplan für Primärstatus auf Bearbeiten.

      2. Die Seite "Primärzeitplan bearbeiten" wird angezeigt.

    2. Geben Sie in den Bereichen Aktivierung und Verfall Zeiten an, um einen Zeitraum zu bestimmen, innerhalb dessen Packages für den Primärstatus verfügbar sein sollen.
    3. Klicken Sie auf OK, um den Zeitplan zu speichern und zur Seite "Policy bearbeiten" zurückzukehren.

      4. Beachten Sie, dass der Zeitplan jetzt im Abschnitt "Zeitplan für Primärstatus" angezeigt wird.

      Weitere Informationen dazu finden Sie in folgenden Abschnitten:

    4. Festlegen des Primär- und Sekundärzeitplans für Packages
    5. Festlegen des Aktualisierungszeitplans für Packages
    6. Festlegen des Reparaturzeitplans für Packages
  4. Nachdem Sie die Zeitpläne festgelegt haben, überprüfen Sie die Zeitpläne, die auf der Seite "Allgemeine Zeitpläne festlegen" angezeigt werden.

    5. Hinweis: Die Zeitpläne, die auf der Seite "Allgemeine Zeitpläne festlegen" angezeigt werden, werden von allen Packages übernommen, die Sie ausgewählt haben. Dieses Verhalten gilt selbst dann, wenn Sie einen Zeitplantyp nicht bearbeitet oder geändert haben (etwa, wenn Sie zwar den Primärzeitplan, aber nicht den Aktualisierungszeitplan geändert haben).

  5. Klicken Sie nach der Überprüfung der Zeitpläne auf OK, um den Zeitplan zu speichern und zur Seite "Policy bearbeiten" zurückzukehren.

Nachdem Sie die Policy bearbeitet haben, sehen Sie sich die Vorschau an, und speichern Sie die Policy. Weitere Informationen dazu finden Sie unter Erstellen der Vorschau und Speichern von Policy-Änderungen.
