Die Seite "Package-Ansicht" ist eine gute Alternative zur Seite "Zielansicht". Sie können diese Seite verwenden, wenn Sie die Ziele anzeigen möchten, denen ein oder mehrere allgemeine Packages zugewiesen sind. Über diese Seite können Sie auch Ziele hinzufügen, Ziele entfernen oder Policys bearbeiten.

So fügen Sie auf der Seite "Package-Ansicht" Ziele hinzu oder entfernen diese:

  1. Klicken Sie auf die Registerkarte Package-Ansicht.
  2. Wählen Sie links auf der Seite ein oder mehr Packages aus. Weitere Informationen dazu finden Sie in folgenden Abschnitten:
    • Anzeigen von Packages
    • Suchen nach Packages
    • Anzeigen von Zielen, denen ein Package zugewiesen wurde

      • Jedes Mal, wenn Sie ein Package auswählen, wird dieses oben rechts auf der Seite in der Liste der ausgewählten Packages angezeigt. Um ein Package aus dieser Liste zu entfernen, klicken Sie daneben auf das Symbol "Entfernen" .

      Unter der Liste der ausgewählten Ziele wird eine Liste der Ziele angezeigt, denen alle ausgewählten Packages (der Schnittmenge der Policys für die ausgewählten Packages) direkt zugewiesen sind. Wenn einem Ziel nicht alle ausgewählten Packages zugewiesen sind, wird es nicht in der Liste angezeigt. Die angezeigten Ziele können die angezeigten Packages bereits erhalten haben oder noch nicht.

  3. So fügen Sie Ziele hinzu:
    1. Klicken Sie auf Hinzufügen.
    2. Verwenden Sie den linken Bereich der Seite, um ein Ziel zu finden und auszuwählen. Weitere Informationen dazu finden Sie unter Anzeigen von Zielen.

      3. Jedes Mal, wenn Sie ein Ziel auswählen, wird dieses rechts auf der Seite in der Liste der ausgewählten Ziele angezeigt.

    3. Geben Sie für jedes Ziel einen Package-Status und -Zeitplan an:
      • Wählen Sie aus den Spalten "Primär- und Sekundärstatus" einen Status für den Package-Status aus.
      • Wählen Sie für den Zeitplan die Ziele aus, auf die der Zeitplan angewendet werden soll, und klicken Sie auf Allgemeinen Zeitplan festlegen.

        • Weitere Informationen dazu finden Sie unter Festlegen von Status und Zeitplänen für Packages in einer Policy.

    4. Klicken Sie nach dem Hinzufügen der Ziele auf Hinzufügen.
  4. So entfernen Sie Ziele:
    1. Wählen Sie ein oder mehr Ziele aus, deren allgemeine Packages entfernt werden sollen.
    2. Klicken Sie auf Entfernen.
    3. Klicken Sie auf der Bestätigungsseite auf Entfernen.

Nach dem Hinzufügen oder Entfernen der Ziele werden Sie zur Seite "Package-Ansicht" zurückgeleitet.

So bearbeiten Sie Policys von der Seite "Package-Ansicht" aus:

  1. Klicken Sie auf die Registerkarte Package-Ansicht.
  2. Wählen Sie links auf der Seite ein oder mehr Packages aus. Weitere Informationen dazu finden Sie in folgenden Abschnitten:
    • Anzeigen von Packages
    • Suchen nach Packages
    • Anzeigen von Zielen, denen ein Package zugewiesen wurde

      • Jedes Mal, wenn Sie ein Package auswählen, wird dieses oben rechts auf der Seite in der Liste der ausgewählten Packages angezeigt. Um ein Package aus dieser Liste zu entfernen, klicken Sie daneben auf das Symbol "Entfernen" .

      Unter der Liste der ausgewählten Ziele wird eine Liste der Ziele angezeigt, denen alle ausgewählten Packages (der Schnittmenge der Policys für die ausgewählten Packages) direkt zugewiesen sind. Wenn einem Ziel nicht alle ausgewählten Packages zugewiesen sind, wird es nicht in der Liste angezeigt. Die angezeigten Ziele können die angezeigten Packages bereits erhalten haben oder noch nicht.

  3. Klicken Sie auf Bearbeiten.
  4. Nehmen Sie die Änderungen an der Policy vor. Weitere Informationen dazu finden Sie unter Allgemeine Anweisungen zum Erstellen und Bearbeiten von Policys.

Nach dem Bearbeiten der Policy werden Sie zur Seite "Package-Ansicht" zurückgeleitet.
