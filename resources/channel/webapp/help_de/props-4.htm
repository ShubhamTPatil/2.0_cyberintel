In diesem Abschnitt wird beschrieben, wie Package-Eigenschaften für Packages festgelegt werden. Package-Eigenschaften stellen Informationen über das Package sowie über dessen Ausführung zur Verfügung. Weitere Informationen dazu finden Sie unter Was sind Package-Eigenschaften?.

Hinweis: Sie können Einstellungen auf Zielebene, wie die Tuner- oder Package-Eigenschaften, nur für jeweils ein Ziel festlegen. Sie können keine Einstellungen auf Zielebene einrichten, wenn Sie sich im Mehrfachauswahlmodus befinden und gleichzeitig Policys für mehrere Ziele bearbeiten. Sie können jedoch für alle Ziele Einstellungen auf Zielebene einrichten, indem Sie das Ziel Alle Clients auswählen.

Bei diesem Vorgang wird davon ausgegangen, dass Sie sich auf der Seite "Policy bearbeiten" auf der Registerkarte "Packages" befinden. Weitere Informationen dazu finden Sie unter Allgemeine Anweisungen zum Erstellen und Bearbeiten von Policys.

So legen Sie Package-Eigenschaften für ein Ziel fest:

  1. Klicken Sie auf der Seite "Policy bearbeiten" auf die Registerkarte Erweitert > Tuner/Package-Eigenschaften.

    2. Die Seite "Tuner- und Package-Eigenschaften" wird angezeigt.

  2. Führen Sie zum Festlegen der Package-Eigenschaften in der Tabelle "Package-Eigenschaften" die folgenden Schritte aus:
    1. Wählen Sie aus der Liste Package das Package oder die Gruppe von Packages aus, für die Sie Eigenschaften festlegen möchten. Wählen Sie eine der folgenden Optionen aus:
      • URL-Adresse eines bestimmten Package: Wählen Sie diese Option aus, wenn Sie lediglich die Eigenschaft eines bestimmten Packages festlegen möchten.
      • Alle Packages: Wählen Sie diese Option aus, wenn Sie die Eigenschaft für alle Packages auf den Client-Tunern, außer für den Policy Service-Channel, festlegen möchten.
      • Alle Packages mit Subscription: Wählen Sie diese Option aus, wenn Sie die Eigenschaft für alle Packages festlegen möchten, die Teil der Policy sind, außer wenn diese den Status Deinstallieren, Ausschließen, Primär oder Ankündigen besitzen. Auch der Policy Service-Channel ist nicht betroffen.
      • Policy Service Channel: Wählen Sie diese Option aus, wenn Sie die Eigenschaft lediglich für den Policy Service-Channel festlegen möchten.
    2. Geben Sie den Name und Wert der Eigenschaft ein.
    3. Klicken Sie auf Übernehmen.

      4. Die Eigenschaft wird dem Textfeld "Eigenschaften" hinzugefügt.

      Hinweis: Sie können die URL-Adresse des Package sowie den Namen und den Wert der Package-Eigenschaft auch direkt in das Textfeld Eigenschaften eingeben. Verwenden Sie das unter Tuner- und Package-Eigenschaften, Format beschriebene Format.

  3. Überprüfen Sie die im Textfeld Eigenschaften aufgelisteten Eigenschaften.

    4. Hinweis: Durch Löschen einer Eigenschaft aus dem Textfeld Eigenschaften wird diese nicht von den Zielclients entfernt. Daher bleibt die Eigenschaft auf dem Ziel-Tuner bestehen (nicht von Policy Management verwaltet). Eine Anleitung zum Löschen von Eigenschaften finden Sie unter Löschen von Tuner- und Package-Eigenschaften.

Nachdem Sie die Policy bearbeitet haben, sehen Sie sich die Vorschau an, und speichern Sie die Policy. Weitere Informationen dazu finden Sie unter Erstellen der Vorschau und Speichern von Policy-Änderungen.
