Eine Collection ist eine aus einer Datenbank- oder LDAP-Abfrage resultierende Liste mit Benutzer- oder Computernamen.

Auf Basis einer LDAP-Abfrage. Zum Erstellen einer Collection auf Basis einer LDAP-Abfrage erstellen Sie über die Befehlszeileneingabe von Policy Manager eine Abfrage, die eine Liste der Benutzer und Computer ausgibt, die Ihren Kriterien entsprechen. Anschließend aktualisieren Sie die Abfrage entweder manuell oder entsprechend eines Zeitplans. Die sich daraus ergebende Liste der Benutzer und Computer steht als Ziel in Policy Manager zur Auswahl bereit.

Auf Basis einer Datenbankabfrage. Wenn Sie das BMC CM-Inventory-Modul verwenden, können Sie mit Report Center Collections erstellen, die Sie in Policy Manager als Ziele verwenden können. Zum Erstellen einer Collection verwenden Sie Report Center und erstellen eine Abfrage, die eine Liste der Ihren Kriterien entsprechenden Computer ausgibt. Nachdem Sie die Abfrage im Ordner "Collections" (in Report Center) gespeichert haben, können Sie diese entweder manuell oder nach einem Zeitplan ausführen.

Sie können der gesamten Collection oder einzelnen Elementen der Collection Policys zuweisen. Den Gruppen des Ergebnissatzes können jedoch keine Policys zugewiesen werden. Es handelt sich um temporäre Gruppen, denen keine Policys zugewiesen werden können.

Eine Anleitung zum Erstellen, Ausführen und Löschen von Datenbankabfrage-Collections finden Sie im Administratorhandbuch für Report Center (Report Center Administrator's Guide). Eine Anleitung zum Erstellen, Ausführen und Löschen von LDAP-Abfrage-Collections finden Sie im Administratorhandbuch für Policy Management.. Dieses Handbuch enthält auch Informationen darüber, wie Collections im Verzeichnisdienst gespeichert sind. Beide Handbücher sind auf der BMC Customer Support-Website verfügbar.
