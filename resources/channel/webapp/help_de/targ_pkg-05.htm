Sie kennzeichnen ein ausgeschlossenes Ziel, indem Sie für ein Ziel-Package-Paar den Status Ausschließen angeben. Der Status "Ausschließen" gibt an, dass das Ziel das zugeordnete Package nicht erhält, auch wenn es an anderer Stelle zugewiesen wurde, z. B. in einer Zielgruppe. Diese Funktion ist sinnvoll, wenn alle Mitglieder einer großen Zielgruppe wie "Alle Clients", mit Ausnahme einiger weniger, ein Package erhalten sollen. In diesem Fall weisen Sie das Package der Zielgruppe zu, erstellen separate Policys für die auszuschließenden Ziele und legen für diese Ziele den Status "Ausschließen" fest.
