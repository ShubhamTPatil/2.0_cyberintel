Bei einem Ziel spricht man von einer direkten Zuweisung, wenn ein Package oder eine Eigenschaft dem Ziel als Teil einer Policy zugewiesen wurde. Wenn ein Ziel einem Package nur deshalb zugewiesen wurde, weil es zu einer Benutzer- oder Computergruppe gehört, der das Package zugewiesen wurde, handelt es sich um eine indirekte Zuweisung.

Beim Anzeigen von Zielen sind direkt zugewiesene Ziele in der Browser-basierten Benutzeroberfläche mit einem Policy-Symbol gekennzeichnet.

Hinweis: Wenn Sie alle Packages und Eigenschaften löschen, die einem Ziel direkt zugewiesen sind, wird das Ziel nicht mehr mit einem Policy-Symbol gekennzeichnet. Das Policy-Symbol wird weiterhin angezeigt, wenn andere Tuner- oder Package-Eigenschaften dem Ziel direkt zugewiesen sind (auch wenn alle Packages gelöscht wurden).
