Im linken Bereich der Zielansicht werden die Ziele angezeigt, denen Sie Packages zuweisen können. Wenn Sie ein Ziel auswählen, werden im rechten Bereich der Seite die dem Ziel direkt und indirekt zugewiesenen Packages angezeigt.

Durch Navigieren im linken Bereich der Seite können Sie alle verfügbaren Ziele anzeigen. Im linken Bereich der Seite wird eine Liste der Ziele angezeigt: einzelne Benutzer und Computer, Benutzer- und Computergruppen sowie Collections, die über eine Datenbank- bzw. LDAP-Abfrage ermittelt wurden.

Wenn Sie die Seite zum ersten Mal besuchen, wird die Home-Ansicht des Ziels angezeigt. Sie besteht aus den übergeordneten Containern des Verzeichnisdienstes, in dem die Zielinformationen gespeichert sind. In Active Directory werden in der Regel die Stammdomänen der Gesamtstrukturumgebung angezeigt. Sie können die Stammdomänen erweitern, um die Objekte anzuzeigen, die in dieser Domäne als Ziele verwendet werden können. Weitere Informationen dazu finden Sie im Abschnitt über Active Directory im Administratorhandbuch für Policy Management, das auf der BMC Customer Support-Website verfügbar ist.

Die Zielsymbole auf der Seite "Zielansicht" zeigen an, wie Policy Manager die Liste der Ziele abruft:

    • Domänen: Ein erweiterbares Domänensymbol zeigt an, dass die Ziele aus einer Active Directory-Gesamtstrukturumgebung abgerufen werden. Das Symbol steht für Stammdomänen und untergeordnete Domänen.
    • Container: Ein erweiterbares Ordnersymbol weist darauf hin, dass die Ziele vom Verzeichnisdienst abgerufen werden. Das Symbol steht für Organisationseinheiten und Container.
    • Benutzer (bezogen vom Transmitter): Ein erweiterbares Transmitter-Symbol weist darauf hin, dass Benutzer und Benutzergruppen von dem Transmitter abgerufen werden, auf dem Sie das Policy Service-Plugin veröffentlicht haben (über die Seite "Plugin-Konfiguration").

      • Hinweis: Wenn Sie (oder ein Primäradministrator) das Policy Service-Plugin noch nicht auf dem Transmitter veröffentlicht haben, erhält Policy Manager keine Anweisungen, welcher Transmitter verwendet werden soll. Policy Manager kann in diesem Fall keine Benutzer- und Benutzergruppeninformationen abrufen. Wenn der Zugriff auf den Transmitter beschränkt ist, müssen Sie außerdem den Benutzernamen und das Passwort für die Authentifizierung beim Transmitter einrichten. Führen Sie dazu folgende Befehlszeilenoption aus:

      -txadminacess <Benutzername> <Passwort>

      Mit diesem Befehl legen Sie den Benutzernamen und das Passwort für den Transmitter fest. Sie müssen dennoch den Benutzernamen und das Passwort für die Authentifizierung bei Policy Manager angeben.

      Weitere Informationen zu den Befehlszeilenoptionen finden Sie im Abschnitt über Befehlszeilen im Administratorhandbuch für Policy Management, das auf der BMC Customer Support-Website verfügbar ist.

Zusätzlich wird das Ziel "Alle Clients" angezeigt. Dabei handelt es sich um ein virtuelles Ziel, das alle Clients an Ihrem Standort enthält, auf denen Policy Service installiert ist.

Wenn Sie eine Gesamtstrukturumgebung mit mehreren Domänen (Active Directory) anzeigen, können Sie verschiedene Strukturen und Domänen übergreifend durchsuchen.
