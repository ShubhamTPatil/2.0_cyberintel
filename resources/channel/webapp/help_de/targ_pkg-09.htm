In diesem Abschnitt wird beschrieben, wie Sie auf der Seite "Zielansicht" den Inhalt eines erweiterbaren Ziels (z. B. Benutzer in einer Benutzergruppe) anzeigen können.

So zeigen Sie den Inhalt eines erweiterbaren Ziels auf der Seite "Zielansicht" an:

    • Klicken Sie auf der Seite "Zielansicht" neben dem Namen des erweiterbaren Ziels auf "+".

Tipp: Wenn Sie den Mauszeiger auf einem Ziel platzieren, wird jeweils sein Distinguished Name (DN) oder die Organisationseinheit (OU) im Verzeichnisdienst angezeigt.

Wenn Sie ein Ziel erweitert haben, um dessen Mitglieder anzuzeigen, sehen Sie im linken Bereich der Seite "Zielansicht" folgende Änderungen:

    • Wenn Sie einen Container erweitern, ändert sich der Titel im rechten Bereich der Seite und zeigt den Container an. Der Titel wird so lange anzeigt, bis Sie zu einem anderen Container wechseln.
    • Wenn Sie eine Gruppe erweitern, werden die Mitglieder der Gruppe angezeigt und der Name der Gruppe wird der Pfadangabe in der Zielhierarchie hinzugefügt.
    • Oben rechts auf der Seite werden Suchoptionen angezeigt, über die Sie ein Ziel innerhalb des aktuellen Containers suchen können. Weitere Informationen dazu finden Sie unter Suchen nach Zielen.
    • Unter den Suchoptionen wird ein Link angezeigt, mit dem Sie zwischen Einzel- und Mehrfachauswahlmodus wechseln können.
    • Die unter dem Auswahlmoduslink angezeigte Pfadangabe zeigt Ihren Standort in der Container- und Gruppenhierarchie an. So können Sie auf einfache Weise durch die Hierarchie navigieren. Die Pfadangabe zeigt Ihre aktuelle Position in der Container-Hierarchie an. Bei den unterstrichenen Elementen handelt es sich um Links, über die Sie schnell an eine andere Position in der Hierarchie zurückkehren können. Das auf den Link "Home" folgende Element zeigt die Container der höchsten Ebene. Es handelt sich normalerweise um die Stammdomäne.
    • Wenn die erweiterte Liste mehr als eine vorgegebene Anzahl an Elementen enthält, wird der Link Weiter aktiviert, so dass Sie Zugriff auf weitere Elemente in der Liste erhalten. Mit Weiter und Zurück können Sie sich in großen Listen vor und zurück bewegen. In sehr langen Listen können Sie die nummerierte Dropdownliste verwenden, um schneller zu navigieren.

      • Hinweis: Solange Sie nicht alle Suchergebnisse durchgesehen haben, kann Active Directory bzw. ADAM nicht die Gesamtzahl der Einträge bestimmen. Aus diesem Grund wird in der Dropdownliste vor dem Erreichen der letzten Seite für die Gesamtzahl der Einträge "..." angezeigt.

    • Wenn eine Gruppe ungültige Mitglieder enthält, werden diese mit einem Fragezeichen angezeigt und ihre DNs werden abgeblendet und in kursiver Schrift angezeigt. Wenn Sie mit dem Mauszeiger auf das Fragezeichen gehen, werden Informationen zu dem ungültigen Mitglied angezeigt, wie in Abbildung 3-2 dargestellt.
