Zum Suchen nach Zielen stehen Ihnen zwei Optionen zur Verfügung:

    • Einfache Suche. Mit dieser Option können Sie über den Common Name (CN) nach Zielen suchen. Durch die Angabe eines Typs können Sie die Suche eingrenzen. Weitere Informationen dazu finden Sie unter Einfache Suche nach Zielen.
    • Erweiterte Suche. Fortgeschrittenen Benutzern, die mit LDAP-Abfragen und Suchfiltern vertraut sind, steht mit dieser Option eine flexiblere Zielsuchfunktion zur Verfügung. Weitere Informationen dazu finden Sie unter Erweiterte Suche nach Zielen.

Sie können in der derzeit erweiterten Zielgruppe oder im erweiterten Container nach Zielen suchen. Diese Funktion ist dann nützlich, wenn Sie Container verwenden, die eine große Anzahl Untercontainer oder Organisationseinheiten mit vielen Mitgliedern enthalten.

Hinweis: Solange Sie nicht alle Suchergebnisse durchgesehen haben, kann Active Directory bzw. ADAM nicht die Gesamtzahl der Einträge bestimmen. Aus diesem Grund wird in der Dropdownliste vor dem Erreichen der letzten Seite für die Gesamtzahl der Einträge "..." angezeigt.

Einfache Suche nach Zielen

So führen Sie eine einfache Suche nach Zielen durch:

  1. Klicken Sie im linken Bereich der Seite "Zielansicht" auf den Container, in dem Sie suchen möchten.
  2. Klicken Sie im linken Bereich der Seite "Zielansicht" auf "Einfache Suche".

    3. Der Bereich Einfache Suche wird angezeigt.

  3. Geben Sie folgende Informationen an:
    • Geben Sie im Textfeld Suche nach den allgemeinen Namen (CN) vollständig oder teilweise ein.

      • Sie müssen nicht den vollständigen Namen des Ziels kennen. Ein Sternchen (*) kann als Platzhalter für mehrere Zeichen im Namen verwendet werden. Wenn das Sternchen Teil des Namens ist, müssen Sie Escape-Zeichen für das Sternchen verwenden, damit es nicht als Platzhalter verwendet wird. Weitere Informationen dazu finden Sie unter Sonderzeichen in Suchbegriffen.

      Hinweis: Bei der Suche wird Groß- und Kleinschreibung nicht berücksichtigt.

    • Wählen Sie in der Liste Begrenzen auf den Zieltyp aus, nach dem Sie suchen möchten, oder wählen Sie Alle Typen aus.
    • Um die Suche weiter einzugrenzen, aktivieren Sie das Kontrollkästchen Untergeordnete Container nicht durchsuchen.
  4. Klicken Sie auf Start.

Die Suchergebnisse werden unterhalb des Suchbereichs angezeigt. Beim Anzeigen der Ergebnisse in der Liste ändert sich die Pfadangabe, um anzuzeigen, dass es sich um Suchergebnisse handelt.

Um zur obersten Ebene der Zielansicht zurückzukehren, klicken Sie auf den Link Home.

Um zur Startansicht der Suche zurückzukehren, klicken Sie auf die Registerkarte Zielansicht.

Sonderzeichen in Suchbegriffen

Erweiterte Suche nach Zielen

Verwenden Sie die erweiterte Suche, wenn Sie mit den LDAP-Abfragen und Suchfiltern des verwendeten Verzeichnisdiensts vertraut sind. Da die Abfrage von Policy Manager direkt an den Verzeichnisdienst übergeben wird, ist die erweiterte Suche schneller als die einfache Suche. Sie können einfache Abfragen eingeben, wie die Suche nach Zielen, die users im CN enthalten:

cn=users

Für etwas komplexere Abfragen können Sie bestimmte Kriterien eingeben, indem Sie z. B. zum Suchen nach Zielen, die im CN entweder users oder groups enthalten, die Schreibweise für or (|) verwenden:

(|(cn=users) (cn=groups))

Weitere Informationen zu LDAP-Suchfiltern finden Sie unter http://www.ietf.org/rfc/rfc2254.txt.

Hinweis: Die erweiterte Suche eignet sich nicht, wenn Sie anhand des vollständigen Distinguished Name (DN) nach Zielen suchen möchten.

So führen Sie eine erweiterte Suche nach Zielen durch:

  1. Klicken Sie im linken Bereich der Seite "Zielansicht" auf den Container, in dem Sie suchen möchten.
  2. Klicken Sie im linken Bereich der Seite "Zielansicht" auf "Erweiterte Suche".

    3. Der Bereich Erweiterte Suche wird angezeigt.

  3. Geben Sie im Textfeld LDAP-Abfrage einen LDAP-Suchfilter ein.

    4. Sie können einfache oder komplexe Abfragen eingeben, die von Policy Manager direkt an den Verzeichnisdienst übergeben werden. Weitere Informationen zur Verwendung von Sonderzeichen in einer Abfrage (z. B. *, (, ), \) finden Sie unter Sonderzeichen in Suchbegriffen.

    Hinweis: Bei der Suche wird Groß- und Kleinschreibung nicht berücksichtigt.

  4. Um die Suche weiter einzugrenzen, aktivieren Sie das Kontrollkästchen Untergeordnete Container nicht durchsuchen.
  5. Klicken Sie auf Start.

Die Suchergebnisse werden unterhalb des Suchbereichs angezeigt. Beim Anzeigen der Ergebnisse in der Liste ändert sich die Pfadangabe, um anzuzeigen, dass es sich um Suchergebnisse handelt.

Um zur obersten Ebene der Zielansicht zurückzukehren, klicken Sie auf den Link Home.

Um zur Startansicht der Suche zurückzukehren, klicken Sie auf die Registerkarte Zielansicht.

Sonderzeichen in Suchbegriffen

Bestimmte Zeichen erfordern eine besondere Codierung, wenn diese in Suchbegriffen verwenden werden; egal ob in der Browser-basierten Schnittstelle oder in der Befehlszeile. Um diese Zeichen in Suchbegriffen zu verwenden, müssen Sie für das Zeichen den hexadezimalen ASCII-Code mit vorangestelltem Schrägstrich eingeben. Die Sonderzeichen sind in Tabelle 3-1 aufgeführt.
