Auf der Seite "Zielinformationen" werden alle Packages, die einem oder mehreren ausgewählten Zielen zugewiesen sind, sowie die Installationspriorität, Primär- und Sekundärstatus und Zeitpläne für die einzelnen Ziele angezeigt.

So zeigen Sie Informationen zu einem Ziel an (oder mehreren Zielen, wenn der Mehrfachauswahlmodus aktiviert ist):

  1. Wählen Sie auf der Seite "Zielansicht" ein oder (im Mehrfachauswahlmodus) mehrere Ziele aus.
  2. Klicken Sie auf Detailansicht, um Details zu den Packages anzuzeigen, die den Zielen zugewiesen sind.

    3. Die Seite "Zielinformationen" wird angezeigt. Auf dieser Seite werden alle Packages, die einem oder mehreren ausgewählten Zielen zugewiesen sind, sowie die Installationspriorität, Primär- und Sekundärstatus und Zeitpläne für die einzelnen Ziele angezeigt.

    Sie können Details zu den ausgewählten Packages anzeigen oder deren Zuweisung zu Zielen bearbeiten. Klicken Sie auf den Spaltennamen, um den Spalteninhalt zu sortieren. Lange Listen können durch Klicken auf Zurück und Weiter durchblättert werden.

  3. Um zur obersten Ebene der Seite "Zielansicht" zurückzukehren, klicken Sie auf den Link Basisansicht.

Sortieren der Packages-Liste

Sortieren der Packages-Liste

Auf der Seite "Zielinformationen" können Sie die Liste der Packages nach folgenden Kriterien sortieren:

    • Zeitpläne
    • Name
    • Installationspriorität (nur im Einzelauswahlmodus verfügbar)
    • Ziel, dem das Package direkt zugewiesen ist (nur im Einzelauswahlmodus verfügbar)
    • Typ (wie durch das Symbol angezeigt; z. B. für ein Package und für eine Patch-Gruppe)

So sortieren Sie die Liste der Packages:

Klicken Sie auf der Seite "Zielinformationen" auf den Spaltennamen, um die Packages nach der entsprechenden Spalte zu sortieren (aufsteigend oder absteigend).

    • Wenn Sie nach einer der Zeitplanspalten sortieren, werden die Packages nach Aktivierungsdatum und -zeit für den gewählten Zeitplan sortiert. Wenn Sie beispielsweise nach dem Primärzeitplan sortieren, werden die Packages nach Aktivierungsdatum und -zeit sortiert, die für den Primärzeitplan festgelegt sind.
    • Wenn Sie nach Name des Package sortieren, werden die Packages alphabetisch sortiert.
    • Wenn Sie nach Installationspriorität sortieren, werden die Packages nach der Priorität sortiert, mit der sie installiert werden. Informationen zum Ändern der Priorität für die Installation von Applikationen finden Sie unter Festlegen der Installationspriorität für Packages in einer Policy.
    • Wenn Sie nach dem Ziel sortieren, dem das Package direkt zugewiesen ist, werden zuerst die direkt zugewiesenen Packages angezeigt, gefolgt von den indirekt zugewiesenen.
