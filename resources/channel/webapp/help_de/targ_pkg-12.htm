Rechts auf der Seite "Zielansicht" sind die Packages angezeigt, die einem oder mehreren in der Zielliste ausgewählten Zielen zugewiesen sind. Die Anzeige ist vom gewählten Modus abhängig. Um den Modus zu wechseln, klicken Sie oben in der Zielliste auf Einzeln oder auf Mehrfach. In den folgenden Abschnitten werden die beiden Modi beschrieben:

    • Einzelauswahlmodus
    • Mehrfachauswahlmodus

Einzelauswahlmodus

Im Einzelauswahlmodus können Sie ein einzelnes Ziel auswählen und die Packages anzeigen, die diesem direkt oder indirekt zugewiesen sind.

So zeigen Sie die Packages an, die einem einzelnen Ziel zugewiesen sind:

  1. Stellen Sie auf der Seite "Zielansicht" sicher, dass Sie den Einzelauswahlmodus aktiviert haben. Wenn Sie sich im Mehrfachauswahlmodus befinden, klicken Sie auf Einzeln, um in den Einzelauswahlmodus zu wechseln.
  2. Wählen Sie links auf der Seite "Zielansicht" das Ziel aus, für das Sie Packages anzeigen möchten.

    3. Wenn Sie auf ein Ziel klicken, werden alle bereits zugewiesenen Packages im rechten Bereich der Seite angezeigt. Das ausgewählte Ziel kann die angezeigten Packages bereits erhalten haben oder noch nicht.

    Hinweis: Es erfolgt keine Auflösung der Statuskonflikte, daher ist es möglich, dass zwei Packages mit demselben Namen (oder URL), aber anderem Status angezeigt werden.

Im rechten Bereich der Seite wird oben der Name des ausgewählten Ziels angezeigt. (Gewöhnlich das cn-Attribut des Eintrags im Verzeichnisdienst.) Unterhalb des Zielnamens befindet sich eine Tabelle, in der alle dem ausgewählten Ziel (direkt und indirekt) zugewiesenen Packages aufgeführt sind.

Im Einzelauswahlmodus stehen Ihnen folgende Optionen zur Verfügung:

Detailansicht: Status und Zeitpläne der einzelnen, dem ausgewählten Ziel zugewiesenen Packages anzeigen.

Bearbeiten: Policy für das Ziel erstellen oder bearbeiten. Weitere Informationen dazu finden Sie unter Erstellen und Bearbeiten von Policys.

Hinweis: Wenn Sie eine Policy bearbeiten, werden nur die einem Ziel direkt zugewiesenen Packages angezeigt. Zum Bearbeiten von Policys, die indirekt zugewiesene Packages umfassen, wählen Sie das Ziel aus, dem diese direkt zugewiesen sind. In der Spalte "Direkt zugewiesen zu" wird angezeigt, welchem Ziel die Packages direkt zugewiesen sind. Weitere Informationen zum Konzept der direkten Zuweisungen finden Sie unter Was ist ein direkt zugewiesenes Ziel?.

Die folgenden Spalten werden in der Tabelle im rechten Bereich der Seite angezeigt:

Packages: Zeigt die Namen der Packages an, die dem ausgewählten Ziel zugewiesen sind. Gehen Sie mit dem Mauszeiger über den Package-Namen, um die URL-Adresse anzuzeigen.

Primär- und Sekundärstatus: Zeigt den oder die Status an, der bzw. die die Art der Verteilung des Package an die einzelnen Clients bestimmt. Eine Liste der Installationsstatus finden Sie unter Übersicht über Installationsstatus.

Direkt zugewiesen zu: Zeigt das Ziel an, dem das Package direkt zugewiesen wurde. Das Package wird in der Liste aufgeführt, weil das ausgewählte Ziel Mitglied einer Gruppe (oder des Ziels selbst) ist, der das Package direkt zugewiesen wurde.

Mehrfachauswahlmodus

Im Mehrfachauswahlmodus können Sie zwei oder mehr Ziele auswählen und deren gemeinsame Packages anzeigen.

So zeigen Sie die Packages an, die mehreren Zielen zugewiesen sind:

  1. Wenn Sie sich nicht bereits im Mehrfachauswahlmodus befinden, wechseln Sie links auf der Seite "Zielansicht" in den Mehrfachauswahlmodus, indem Sie auf Mehrfach klicken.
  2. Wählen Sie im linken Bereich der Seite "Zielansicht" die Ziele aus, für die Sie Packages anzeigen möchten.

    3. Jedes Mal, wenn Sie ein Ziel auswählen, wird dieses oben rechts auf der Seite in der Liste der ausgewählten Ziele angezeigt. Um ein Ziel aus dieser Liste zu entfernen, klicken Sie daneben auf das Symbol "Entfernen" .

    Unter der Liste der ausgewählten Ziele wird eine Liste der Packages angezeigt, die allen ausgewählten Zielen (der Schnittmenge der Policys für die ausgewählten Ziele) direkt zugewiesen sind. Wenn ein Package nicht allen ausgewählten Zielen zugewiesen ist, wird es nicht in der Liste angezeigt. Das ausgewählte Ziel kann die angezeigten Packages bereits erhalten haben oder noch nicht.

Im Mehrfachauswahlmodus stehen Ihnen folgende Optionen zur Verfügung:

Detailansicht: Status und Zeitpläne der den ausgewählten Zielen zugewiesenen Packages anzeigen.

Bearbeiten: Policy für die Ziele erstellen oder bearbeiten. Weitere Informationen dazu finden Sie unter Erstellen und Bearbeiten von Policys. Beim Bearbeiten mehrerer Ziele können Sie nur Packages hinzufügen, bearbeiten und entfernen, die allen Zielen zugewiesen sind.

Die folgenden Spalten werden in der Tabelle im rechten Bereich der Seite angezeigt:

Packages: Zeigt die Namen der Packages an, die den ausgewählten Zielen zugewiesen sind. Gehen Sie mit dem Mauszeiger über den Package-Namen, um die URL-Adresse anzuzeigen.

Primär- und Sekundärstatus: Zeigt den oder die Status an, der bzw. die die Art der Verteilung des Package an die einzelnen Clients bestimmt. Eine Liste der Installationsstatus finden Sie unter Übersicht über Installationsstatus.
