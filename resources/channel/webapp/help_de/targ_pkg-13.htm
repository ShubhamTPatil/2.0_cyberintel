Wenn Sie Policy Manager verwenden, können Sie Packages auswählen und diese Zielen zuweisen. Ein Package ist normalerweise eine Applikation, die auf Zielen installiert werden soll, kann jedoch aus beliebigen Daten bestehen, die Sie verteilen möchten. Gewöhnlich verwenden Sie BMC CM Application Packager zum Erstellen eines Package.

Nachdem Sie einem Ziel ein Package zugewiesen haben, können Sie jedem Package-Ziel-Paar einen Status sowie Zeitpläne und Eigenschaftseinstellungen zuweisen, die sich auf die Installation bei den einzelnen Clients auswirken.

Packages sind durch ein Package-Symbol gekennzeichnet.
