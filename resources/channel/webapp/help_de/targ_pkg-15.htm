Um Packages anzuzeigen, die einem oder mehreren Zielen zugeordnet sind, klicken Sie auf die Registerkarte Package-Ansicht. Auf der Seite "Package-Ansicht" werden alle Packages angezeigt, die einem oder mehreren Zielen zugewiesen wurden. (Wenn Sie Packages einer Policy hinzufügen, werden diese auch der Liste hinzugefügt.) Wenn Sie ein Package auswählen, können Sie entsprechende Informationen und die Ziele anzeigen, denen es zugewiesen wurde.

Tipp: Gehen Sie mit dem Mauszeiger über ein Package, um dessen URL anzuzeigen.

Im linken Bereich der Seite "Package-Ansicht" wird eine Liste der Packages angezeigt, die Zielen zugewiesen wurden. In der Liste werden die Packages und die URLs angezeigt, von denen sie abgerufen werden. Um der Liste ein neues Package hinzuzufügen, müssen Sie dieses identifizieren und einem Package zuweisen, indem Sie eine Policy erstellen oder bearbeiten. Weitere Informationen dazu finden Sie unter Erstellen und Bearbeiten von Policys.

    • Standardmäßig werden die Packages alphabetisch nach Package-Name sortiert aufgelistet. Klicken Sie auf Anzeigen: URL, um Package-URLs anstelle von Package-Namen anzuzeigen.
    • Oben rechts auf der Seite wird ein Suchtextfeld angezeigt, über das Sie nach einem Package suchen können. Sie müssen nicht den vollständigen Namen des Package kennen. Sie können ein Sternchen (*) als Platzhalter für mehrere Zeichen im Namen verwenden.

      • Bei der Suche wird Groß- und Kleinschreibung nicht berücksichtigt.

    • Unter dem Suchtextfeld wird ein Link angezeigt, mit dem Sie zwischen Einzel- und Mehrfachauswahlmodus wechseln können. Sie können die Informationen zu einem Package oder zu mehreren gleichzeitig anzeigen. Diese Informationen umfassen alle Ziele, denen das ausgewählte Package zugewiesen ist. Um die Informationen zu vielen Packages gleichzeitig anzuzeigen, wechseln Sie durch Klicken auf Mehrfach in den Mehrfachauswahlmodus.
    • Wenn die Liste mehr als die vorgegebene Anzahl an Elementen enthält, wird der Link Weiter aktiviert, so dass Sie Zugriff auf weitere Elemente in der Liste erhalten. Mit Weiter und Zurück können Sie sich in großen Listen vor und zurück bewegen. In sehr langen Listen können Sie die nummerierte Dropdownliste verwenden, um schneller zu navigieren.
