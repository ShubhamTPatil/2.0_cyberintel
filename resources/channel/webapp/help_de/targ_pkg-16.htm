Sie können nach Packages suchen, die einem oder mehreren Zielen zugewiesen sind. Diese Funktion ist sinnvoll, wenn Sie es mit einer großen Zahl von Packages zu tun haben.

So suchen Sie nach einem Package:

  1. Im linken Bereich der Seite "Package-Ansicht" geben Sie in das Textfeld Suche den Namen des Package ein.

    2. Sie müssen nicht den vollständigen Namen des Package kennen. Ein Sternchen (*) kann als Platzhalter für mehrere Zeichen im Namen verwendet werden. Wenn das Sternchen Teil des Namens ist, müssen Sie Escape-Zeichen für das Sternchen verwenden, damit es nicht als Platzhalter verwendet wird. Weitere Informationen dazu finden Sie unter Sonderzeichen in Suchbegriffen.

    Bei der Suche wird Groß- und Kleinschreibung nicht berücksichtigt.

  2. Klicken Sie auf Start, oder drücken Sie die Eingabetaste auf der Tastatur.

Die Suchergebnisse werden in der Package-Liste angezeigt. Klicken Sie auf Zurücksetzen, um alle zugewiesenen Packages anzuzeigen.

Wenn die Liste mehr als die vorgegebene Anzahl an Elementen enthält, wird der Link Weiter aktiviert, so dass Sie Zugriff auf weitere Elemente in der Liste erhalten. Mit Weiter und Zurück können Sie sich in großen Listen vor und zurück bewegen. In sehr langen Listen können Sie die nummerierte Dropdownliste verwenden, um schneller zu navigieren.

Hinweis: Solange Sie nicht alle Suchergebnisse durchgesehen haben, kann Active Directory bzw. ADAM nicht die Gesamtzahl der Einträge bestimmen. Aus diesem Grund wird in der Dropdownliste vor dem Erreichen der letzten Seite für die Gesamtzahl der Einträge "..." angezeigt.
