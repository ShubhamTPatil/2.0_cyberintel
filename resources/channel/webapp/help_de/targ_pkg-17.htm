Rechts auf der Seite "Package-Ansicht" sind die Ziele angezeigt, denen ein oder mehrere in der Package-Liste ausgewählte Packages zugewiesen wurden. Die Anzeige ist vom gewählten Modus abhängig. Um den Modus zu wechseln, klicken Sie oben in der Package-Liste auf Einzeln oder auf Mehrfach. In den folgenden Abschnitten werden die beiden Modi beschrieben:

    • Einzelauswahlmodus
    • Mehrfachauswahlmodus

Einzelauswahlmodus

Im Einzelauswahlmodus können Sie ein einzelnes Package und die Ziele anzeigen, denen es zugewiesen ist.

So zeigen Sie die Ziele an, denen ein einzelnes Package zugewiesen ist:

  1. Stellen Sie auf der Seite "Package-Ansicht" sicher, dass Sie den Einzelauswahlmodus aktiviert haben. Wenn Sie sich im Mehrfachauswahlmodus befinden, klicken Sie auf Einzeln, um in den Einzelauswahlmodus zu wechseln.
  2. Wählen Sie links auf der Seite "Package-Ansicht" das Ziel aus, für das Sie Packages anzeigen möchten.

    3. Wenn Sie auf ein Package klicken, werden alle Ziele, denen das Package bereits zugewiesen wurde, im rechten Bereich der Seite angezeigt. Die Ziele können das angezeigten Package bereits erhalten haben oder noch nicht.

Im rechten Bereich der Seite wird oben der Name des ausgewählten Package angezeigt. Unter dem Package-Namen befindet sich eine Tabelle, in der alle Ziele aufgeführt sind, denen das ausgewählte Package bereits zugewiesen wurde.

Im Einzelauswahlmodus stehen Ihnen folgende Optionen zur Verfügung:

Detailansicht: Status und Zeitpläne der einzelnen, dem ausgewählten Ziel zugewiesenen Packages anzeigen.

Bearbeiten: Status und Zeitpläne für die Ziele bearbeiten, deren Kontrollkästchen Sie aktiviert haben. Weitere Informationen dazu finden Sie unter Erstellen und Bearbeiten von Policys.

Die folgenden Spalten werden in der Tabelle im rechten Bereich der Seite angezeigt:

Ziele: Zeigt die Namen der Ziele an, die dem ausgewählten Package zugewiesen sind. (Gewöhnlich das cn-Attribut des Eintrags im Verzeichnisdienst.)

Tipp: Wenn Sie die Maus über ein Ziel bewegen, wird dessen Organisationseinheit im Verzeichnisdienst angezeigt.

Primär- und Sekundärstatus: Zeigt den oder die Status an, der bzw. die die Art der Verteilung des Package an die einzelnen Clients bestimmt. Eine Liste der Installationsstatus finden Sie unter Übersicht über Installationsstatus.

Mehrfachauswahlmodus

Im Mehrfachauswahlmodus können Sie zwei oder mehr Packages auswählen und die Ziele anzeigen, denen die Packages zugewiesen sind.

So zeigen Sie die Ziele an, denen mehrere Packages zugewiesen sind:

  1. Wenn Sie sich nicht bereits im Mehrfachauswahlmodus befinden, wechseln Sie links auf der Seite "Package-Ansicht" in den Mehrfachauswahlmodus, indem Sie auf Mehrfach klicken.
  2. Wählen Sie links auf der Seite "Package-Ansicht" die Packages aus, für die Sie Ziele anzeigen möchten.

    3. Jedes Mal, wenn Sie ein Package auswählen, wird dieses oben rechts auf der Seite in der Liste der ausgewählten Packages angezeigt. Um ein Package aus dieser Liste zu entfernen, klicken Sie daneben auf das Symbol "Entfernen" .

    Unter der Liste der ausgewählten Packages wird eine Liste der Ziele angezeigt, denen alle ausgewählten Packages (der Schnittmenge der Policys für die ausgewählten Packages) direkt zugewiesen sind. Wenn einem Ziel nicht alle ausgewählten Packages zugewiesen sind, wird es nicht in der Liste angezeigt. Die angezeigten Ziele können die angezeigten Packages bereits erhalten haben oder noch nicht.

Im Mehrfachauswahlmodus stehen Ihnen folgende Optionen zur Verfügung:

Detailansicht: Status und Zeitpläne der einzelnen, dem ausgewählten Ziel zugewiesenen Packages anzeigen.

Bearbeiten: Status und Zeitpläne für die Ziele bearbeiten, deren Kontrollkästchen Sie aktiviert haben. Auf der Seite "Policy bearbeiten" können Sie Status und Zeitpläne für die ausgewählten Ziele festlegen.

Die folgenden Spalten werden im rechten Bereich der Seite angezeigt:

Ziele: Zeigt die Namen der Ziele an, die dem ausgewählten Package zugewiesen sind. (Gewöhnlich das cn-Attribut des Eintrags im Verzeichnisdienst.)

Tipp: Wenn Sie die Maus über ein Ziel bewegen, wird dessen Organisationseinheit im Verzeichnisdienst angezeigt.

Primär- und Sekundärstatus: Zeigt den oder die Status an, der bzw. die die Art der Verteilung des Package an die einzelnen Clients bestimmt. Eine Liste der Installationsstatus finden Sie unter Übersicht über Installationsstatus.
