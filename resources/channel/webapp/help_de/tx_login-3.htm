In diesem Abschnitt wird beschrieben, wie Berechtigungen für einen Transmitter zu Policy Manager hinzugefügt oder bearbeitet werden. Die folgenden Anweisungen setzen voraus, dass Sie den Benutzernamen und das Passwort für den Transmitter kennen. Wenn Sie nicht über diese Informationen verfügen, wenden Sie sich an die Person, die für die Administration der Transmitter zuständig ist.

Wenn mehrere Benutzer die Seite „Transmitter-Berechtigungen“ für das gleiche Ziel bearbeiten, werden die zuletzt gespeicherten Änderungen übernommen.

Hinweis: Sie können Einstellungen auf Zielebene, wie zum Beispiel die Transmitter-Berechtigungen, nur für jeweils ein Ziel einrichten. Sie können keine Einstellungen auf Zielebene einrichten, wenn Sie sich im Mehrfachauswahlmodus befinden und gleichzeitig Policys für mehrere Ziele bearbeiten. Sie können jedoch für alle Ziele Einstellungen auf Zielebene einrichten, indem Sie das Ziel Alle Clients auswählen.

Bei diesem Vorgang wird davon ausgegangen, dass Sie sich auf der Seite „Policy bearbeiten“ auf der Registerkarte „Packages“ befinden. Weitere Informationen dazu finden Sie unter Allgemeine Anweisungen zum Erstellen und Bearbeiten von Policys.

So fügen Sie Berechtigungen für einen Transmitter hinzu oder bearbeiten diese:

  1. Klicken Sie auf der Seite „Policy bearbeiten“ auf die Registerkarte Transmitter-Berechtigungen.

    2. Die Seite „Transmitter-Berechtigungen“ wird angezeigt.

  2. Führen Sie einen der folgenden Schritte aus:
    • Um Transmitter-Berechtigungen hinzuzufügen, klicken Sie auf Transmitter-Berechtigungen hinzufügen.
    • Um vorhandene Transmitter-Berechtigungen zu bearbeiten, wählen Sie einen Transmitter aus und klicken auf Bearbeiten.

      • Die Seite „Transmitter-Berechtigungen hinzufügen/bearbeiten“ wird angezeigt.

  3. Geben Sie für Hostname:Portnummer den Hostnamen und die Portnummer des Transmitter ein, z. B. transmitter_name:80.
  4. Geben Sie den Benutzernamen und das Passwort ein, die Sie für den Transmitter verwenden möchten. Geben Sie das Passwort zur Bestätigung erneut ein.
  5. Klicken Sie auf OK, um die Transmitter-Berechtigungen zu speichern.

    6. Die hinzugefügten bzw. bearbeiteten Transmitter-Berechtigungen werden in der Liste angezeigt.

Nachdem Sie die Policy bearbeitet haben, sehen Sie sich die Vorschau an, und speichern Sie die Policy. Weitere Informationen dazu finden Sie unter Erstellen der Vorschau und Speichern von Policy-Änderungen.

Der angegebene Benutzername und das Passwort werden verwendet, wenn Ziele eine Package-Subscription auf einem Transmitter einrichten oder ein Package herunterladen.
