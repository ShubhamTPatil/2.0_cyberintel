In diesem Abschnitt wird beschrieben, wie Berechtigungen für einen Transmitter in Policy Manager gelöscht werden. Wenn Sie in Policy Manager Berechtigungen für einen Transmitter gelöscht haben, können Ziel-Tuner Package-Subscriptions auf diesem Transmitter nur einrichten, wenn Benutzername und Passwort angegeben werden.

Hinweis: Sie können Einstellungen auf Zielebene, wie zum Beispiel die Transmitter-Berechtigungen, nur für jeweils ein Ziel einrichten. Sie können keine Einstellungen auf Zielebene einrichten, wenn Sie sich im Mehrfachauswahlmodus befinden und gleichzeitig Policys für mehrere Ziele bearbeiten. Sie können jedoch für alle Ziele Einstellungen auf Zielebene einrichten, indem Sie das Ziel Alle Clients auswählen.

Bei diesem Vorgang wird davon ausgegangen, dass Sie sich auf der Seite „Policy bearbeiten“ auf der Registerkarte „Packages“ befinden. Weitere Informationen dazu finden Sie unter Allgemeine Anweisungen zum Erstellen und Bearbeiten von Policys.

So löschen Sie die Berechtigungen für einen Transmitter:

  1. Klicken Sie auf der Seite „Policy bearbeiten“ auf die Registerkarte Transmitter-Berechtigungen.

    2. Die Seite „Transmitter-Berechtigungen“ wird angezeigt.

  2. Wählen Sie einen Transmitter aus, und klicken Sie auf Löschen.

    3. Die gelöschten Transmitter-Berechtigungen werden nicht mehr in der Liste angezeigt.

Nachdem Sie die Policy bearbeitet haben, sehen Sie sich die Vorschau an, und speichern Sie die Policy. Weitere Informationen dazu finden Sie unter Erstellen der Vorschau und Speichern von Policy-Änderungen.
