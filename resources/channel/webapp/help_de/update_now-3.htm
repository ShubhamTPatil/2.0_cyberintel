Vor der Durchführung sofortiger Policy-Aktualisierungen müssen Sie diese Funktion zunächst aktivieren, wie in diesem Abschnitt beschrieben. Zur Verwendung dieser Funktion müssen Sie auch die Deployment Manager-Integrationseinstellungen (in den Systemeinstellungen) konfigurieren.

Um festzustellen, ob die ACL-Funktion aktiv ist oder nicht, führen Sie den Mauszeiger auf das Statussymbol rechts oben im Konsolenfenster.

So aktivieren oder deaktivieren Sie sofortige Policy-Aktualisierungen:

  1. Klicken Sie auf die Registerkarte Konfiguration.
  2. Klicken Sie auf der Seite „Konfiguration“ auf den Link Erweiterte Optionen.

    3. Die Seite „Erweitert Optionen“ wird angezeigt.

  3. Um diese Funktion zu aktivieren, aktivieren Sie das Kontrollkästchen Policy-Aktualisierungsfunktion aktivieren. Zum Deaktivieren der Funktion deaktivieren Sie das Kontrollkästchen.
  4. Klicken Sie auf OK, um die Änderungen zu speichern und auf die Konfigurationsseite zurückzukehren.
