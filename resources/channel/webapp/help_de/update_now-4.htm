Sie sollten eine sofortige Policy-Aktualisierung durchführen, wenn Applikationen, Patches, Inhalte oder Aktualisierungen unmittelbar an Zielcomputer verteilt werden sollen.

Sie können die Policy-Aktualisierungen nur für Zielcomputer durchführen. Wenn Sie ein kumuliertes Ziel (wie das Ziel „Alle Clients“, eine Domäne, einen Container oder eine Gruppe) auswählen, das Ziele enthält, die keine Computer sind, erhalten nur die Computerziele Policy-Aktualisierungen. Sofortige Policy-Aktualisierungen können nur im Einzelauswahlmodus durchgeführt werden.

Hinweis: Die Beschränkung der Computerziele betrifft nur sofortige Policy-Aktualisierungen. Bei der Ausführung von Policy Service sucht das Programm nach benutzerbasierten oder computerbasierten Policys.

So führen Sie eine sofortige Policy-Aktualisierung durch:

  1. Wählen Sie auf der Seite „Zielansicht“ oder auf der Seite „Zielinformationen“ ein Ziel aus.
  2. Klicken Sie auf Aktualisieren, und wählen Sie eine der folgenden Optionen:
    • Direkt zugewiesene Policy aktualisieren: Wählen Sie diese Option aus, wenn Sie nur die diesem Ziel direkt zugewiesenen Policys aktualisieren möchten, jedoch keine vererbten Policys aufgrund seiner Mitgliedschaft in einer Gruppe, einem Container, einer Domäne oder dem Ziel „Alle Clients“.
    • Alle Policys aktualisieren: Wählen Sie diese Option aus, wenn Sie alle Policys für dieses Ziel aktualisieren möchten, einschließlich der direkt zugewiesenen und sämtlicher an das Ziel vererbten Policys aufgrund seiner Mitgliedschaft in einer Gruppe, einem Container, einer Domäne oder dem Ziel „Alle Clients“.

      • Die Seite „Aktualisierungsvorschau“ wird angezeigt, auf der Sie bestätigen können, dass die Policy(s) für das richtige Ziel aktualisiert werden sollen.

  3. Bestätigen Sie Ihre Auswahl durch Klicken auf Direkt zugewiesene Policy aktualisieren oder auf Alle Policys aktualisieren.

    4. Policy Manager beginnt mit der Aktualisierung der Policy(s) für das angegebene Ziel. Weitere Informationen zur Funktionsweise von Policy-Aktualisierungen finden Sie unter Wie funktioniert die sofortige Policy-Aktualisierung?.

    Die Seite „Status der Policy-Aktualisierung“ wird angezeigt, auf der Sie den Status der Computer des Ziels überwachen können. Die Seite wird automatisch alle zehn Sekunden aktualisiert. Weitere Informationen dazu finden Sie unter Überwachen und Anzeigen des Status einer Policy-Aktualisierung.

  4. Wenn die Policy-Aktualisierungen für alle Computer des angegebenen Ziels abgeschlossen sind, zeigt Policy Manager die Liste der Computer mit der Angabe an, ob die Aktualisierung erfolgreich war.

    5. Bei Computern, auf denen die Aktualisierung fehlgeschlagen ist, können Sie auf den Status Fehlgeschlagen klicken, um die Logs für diesen Computer anzuzeigen. Wahlweise können Sie auch auf „Fehlgeschlagene erneut versuchen“ klicken, um Policy-Aktualisierungen auf den Computern des Ziels erneut zu versuchen, auf denen sie fehlgeschlagen sind.

  5. Klicken Sie auf Fertig, um auf die Seite „Zielansicht“ bzw. „Zielinformationen“ zurückzukehren.

    6. Beachten Sie, dass neben der Schaltfläche Aktualisieren ein Häkchensymbol angezeigt wird, um Ziele zu kennzeichnen, bei denen die Policy-Aktualisierung abgeschlossen ist. Durch Klicken auf Aktualisieren wird die zusätzliche Option Status der letzten Aktualisierung anzeigen unten in der Dropdownliste angezeigt. Weitere Informationen dazu finden Sie unter Überwachen und Anzeigen des Status einer Policy-Aktualisierung.
