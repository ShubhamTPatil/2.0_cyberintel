Alle für Ziele festgelegten Blackout-Perioden werden eingehalten, wenn Sie eine sofortige Policy-Aktualisierung durchführen. Wenn Sie möchten, dass sofortige Policy-Aktualisierungen während einer Blackout-Periode wirksam werden, müssen Sie die folgenden Komponenten von der Blackout-Periode ausnehmen:

    • Policy Service
    • Patch Service (wenn Patch Management installiert ist)
    • Bestimmte Packages, die von der Blackout-Periode ausgenommen sein sollen.

Weitere Informationen dazu finden Sie unter Festlegen der Blackout-Periode für ein Ziel und Ausnehmen von Packages von der Blackout-Periode.
