Policy Manager permite al usuario administrador asignar paquetes a destinos. Estos destinos representan uno o más clientes finales como, por ejemplo, usuarios, máquinas o grupos. Para cada destino, se asigna al paquete un estado, por ejemplo, "descargar", "instalar" o "desinstalar". El estado que especifique se aplica al paquete en cada destino según una planificación establecida. Policy Manager se basa en navegadores y almacena información sobre paquetes y destinos en un servicio de directorios, utilizando el protocolo LDAP.

Este documento está destinado para directores de redes, administradores del sistema, personal del servicio de asistencia y directores de sistemas que despliegan y configuran Policy Manager o lo utilizan para administrar una red de equipos informáticos. Para este documento, se supone que el usuario está familiarizado con los productos de BMC Configuration Management (CM) como, por ejemplo, tuners y transmitters.

Este producto se compone de los componentes siguientes:

    • Channel de Policy Manager que proporciona la interfaz basada en navegador que se utiliza para crear políticas de distribución de paquetes a los destinos, asignar estados a los paquetes y planificar la descarga e instalación de paquetes. Este componente también se puede usar para especificar la configuración. Policy Manager también proporciona una interfaz de línea de comandos para la configuración de algunos parámetros que se usan para distribuir paquetes a destinos.
    • Channel de Common Management Services, que permite configurar los parámetros para todo el sistema que se aplican a Policy Manager y otras aplicaciones web BMC CM de las que pueda disponer.
    • Channel de Policy Service, que es un componente cliente y se encarga de la aplicación de las políticas asignadas al tuner de destino en el que se ejecuta.

Además, necesita disponer de los servicios de directorios siguientes:

    • Microsoft Active Directory (AD)
    • Microsoft Active Directory Application Mode (ADAM)
    • Sun ONE Directory

Para obtener información sobre las versiones de los servicios de directorios componentes, consulte las notas de la versión de Policy Management, disponible en el sitio web de soporte al cliente de BMC.
