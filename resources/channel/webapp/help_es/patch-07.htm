Si elimina un grupo de parches de una política, dicho grupo de parches se eliminará del destino la próxima vez que Policy Service se actualice. Sin embargo, aunque elimine el grupo de parches, esto no implica que los parches de dicho grupo se desinstalen del destino.

Para desinstalar los parches, deberá utilizar Patch Manager para llevar a cabo los pasos siguientes

  1. Edite el grupo de parches.
  2. Especifique la acción de desinstalación para los parches individuales que desea desinstalar.
  3. Vuelva a publicar el grupo de parches.

La próxima vez que se ejecute Patch Service, se llevará a cabo la operación de actualización del grupo de parches y se desinstalarán los parches especificados. Para obtener más información, consulte la Guía del administrador de Patch Management en el sitio web de soporte al cliente de BMC.
