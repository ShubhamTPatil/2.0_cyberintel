Una política consiste en la asociación entre un destino, uno o más paquetes y el suministro planificado de los paquetes a los destinos.

Se tratan los siguientes temas:

  • ¿Qué es una política?
  • Gestión de tareas
  • Instrucciones generales para la creación y edición de políticas
  • Agregar y eliminar paquetes de una política
  • Especificar estados y planificaciones de paquetes de una política
  • Configurar la prioridad de instalación de los paquetes de una política
  • Copiar políticas
  • Eliminar políticas
  • Administración del software, datos y actualizaciones
  • Editar políticas en Vista de paquete
  • Establecer un periodo de exclusión para un destino
  • Especificar la planificación de Policy Service para un destino
  • Definir propiedades de tuner y paquete para un destino
  • Especificar permisos de transmitter para un destino
  • Especificar el perfil de un destino

    • ADVERTENCIA:  Cada vez que inicia una sesión en la consola y comienza a utilizar Policy Manager, se inicia una sesión del navegador. Muchas de las páginas web asociadas a dicha sesión dependerán de que se hayan visitado anteriormente otras páginas que definen las variables de las sesión. Por este motivo, no deberá acceder a las páginas web introduciendo directamente la dirección URL de las mismas ni mediante los favoritos. Asimismo, se recomienda no utilizar los botones Atrás y Adelante durante una sesión. Este tipo de comportamiento es habitual en las aplicaciones que se ejecutan en navegador.
