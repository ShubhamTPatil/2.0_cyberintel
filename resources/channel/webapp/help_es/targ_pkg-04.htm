Una colección es una lista de nombres de usuarios o de máquinas que se obtiene al ejecutar una consulta de base de datos o una consulta LDAP.

Basada en una consulta LDAP. Para crear una colección basada en una consulta LDAP, utilice la interfaz de la línea de comandos de Policy Manager para crear una consulta que devuelva una lista de usuarios o máquinas que se adecue a sus criterios y, a continuación, actualice dicha consulta, ya sea manualmente o por medio de una planificación. La lista resultante de usuarios o de máquinas se podrá seleccionar como destino en Policy Manager.

Basada en una consulta de base de datos. Si dispone del módulo BMC CM Inventory, utilice Report Center para crear las colecciones y, posteriormente, las podrá emplear como destinos en Policy Manager. Para crear una colección, utilice Report Center para originar una consulta que devuelva una lista de máquinas adecuada a sus criterios, guarde esa consulta en la carpeta Colecciones (en Report Center) y, a continuación, ejecútela, ya sea manualmente o por medio de una planificación.

Puede asignar políticas a la colección entera o a elementos individuales de la colección. Sin embargo, no puede asignarla a los grupos originados como resultado de la consulta. Estos grupos originados como resultado de la consulta son temporales y no pueden tener políticas asignadas.

Para obtener instrucciones sobre cómo crear, ejecutar y eliminar colecciones de consultas de bases de datos, consulte la Guía del administrador de Report Center. Para obtener información sobre cómo crear, ejecutar y eliminar colecciones de consultas LDAP, consulte Guía del administrador de Policy Management. Esta guía también incluye detalles de cómo se almacenan las colecciones en el servicio de directorios. Ambas guías están disponibles en el sitio web de soporte al cliente de BMC.
