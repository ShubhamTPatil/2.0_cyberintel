Identificará un destino excluido cuando seleccione el estado excluir para una pareja destino-paquete. El estado Excluir especifica que el destino no recibe el paquete asociado, incluso si éste ha sido asignado a otros destinos como, por ejemplo, un grupo. Esta función es útil cuando usted desee que no todos, sino unos cuantos miembros de un grupo de destino muy grande como, por ejemplo, Todos los clientes finales, reciban un paquete. En este caso, asigne el paquete a un grupo destino, cree políticas separadas para los destinos que desee excluir y configure el estado de exclusión para ellos.
