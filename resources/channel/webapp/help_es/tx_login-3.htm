Este tema describe cómo agregar o editar permisos para un transmitter en Policy Manager. En las instrucciones que siguen a continuación se da por hecho que usted sabe el nombre de usuario y la contraseña requeridas por el transmitter. Pero, si no dispone de esta información, póngase en contacto con la persona responsable de administrar sus transmitters.

Si diversos usuarios editan la página Permisos de transmitter para el mismo destino, se aplican las últimas ediciones guardadas.

Nota: Puede establecer ciertas configuraciones de nivel de destino, como por ejemplo, los permisos de transmitters, sólo para un destino cada vez. Si tiene habilitado el modo de selección múltiple y está editando las políticas de múltiples destinos a la vez, no podrá establecer una configuración de nivel de destino. Sin embargo, podrá establecer una configuración de nivel de destino para todos los destinos seleccionando el destino Todos los clientes finales.

Este procedimiento da por hecho que se encuentra en la ficha Paquetes de la página Editar política. Para obtener más información, consulte Instrucciones generales para la creación y edición de políticas.

Para agregar o editar los permisos de un transmitter

  1. En la página Editar política, haga clic en la ficha Opciones avanzadas > Permisos de transmitter.

    2. Aparece la página Permisos de transmitter.

  2. Haga algo de lo que se le expone a continuación:
    • Para agregar los permisos de transmitter, haga clic en Agregar permiso de transmitter.
    • Para editar los permisos ya existentes de transmitter, seleccione un transmitter y haga clic en Editar.

      • Aparece la página Agregar/editar permisos de transmitter.

  3. Para los datos de Nombre de host:Número de puerto, introduzca el nombre de host y el número de puerto de transmitter, por ejemplo: transmitter_name:80.
  4. Introduzca el nombre de usuario y la contraseña que desee utilizar con el transmitter. Confirme la contraseña introduciéndola de nuevo.
  5. Haga clic en Aceptar para guardar los permisos de transmitter.

    6. Los permisos agregados o editados de transmitter aparecen en la lista.

Cuando haya terminado de editar la política, visualice los cambios efectuados en Vista previa y guárdela. Para obtener más información, consulte Visualizar previamente y almacenar cambios en las políticas.

El nombre de usuario y la contraseña especificadas se utilizan en el momento de suscribir los destinos o descargar un paquete desde el transmitter.
