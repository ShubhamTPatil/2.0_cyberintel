Este capítulo describe cómo se puede utilizar Policy Manager con la aplicación BMC Deployment Manager para actualizar políticas de forma inmediata en las máquinas de destino.

Se tratan los siguientes temas:

  • Prerrequisitos para la integración con Deployment Manager
  • Habilitar y deshabilitar actualizaciones inmediatas de política
  • Realizar una actualización inmediata de política
  • Supervisar y visualizar del estado de una actualización de política
  • ¿Cómo funciona la actualización inmediata de una política?
  • Periodos de exclusión y actualizaciones inmediatas de políticas
