Les options de ligne de commande de Policy Manager constituent un sous-ensemble des fonctions disponibles dans l'interface gérée par navigateur. Pour consulter la liste de ces options de ligne de commande, reportez-vous au Guide de l’administrateur de Policy Manager, disponible sur le site Web de l’assistance à la clientèle BMC. Les procédures décrites dans la documentation Policy Manager supposent que vous utilisez l'interface gérée par navigateur.

Remarque : si vous utilisez l’interface gérée par navigateur, n’utilisez pas le bouton Précédent et n’accédez pas directement aux pages via des signets ou des URL.
