La procédure d'accès à la documentation relative à BMC Configuration Management (CM) a été modifiée.

Elle se trouve désormais sur le site Web de l’assistance à la clientèle BMC. Pour y accéder, vous devez disposer d'un nom de connexion et d'un mot de passe d'assistance à la clientèle. La recherche de documents s'effectue par nom de produit, version, langue ou type de document. Vous pouvez restreindre ou élargir votre recherche au gré de vos besoins. Vous pouvez ainsi rechercher uniquement les documents de version 6.0.3 ou les documents Policy Management version 6.0.3. Moins vous spécifiez de critères de recherche, plus le nombre de résultats renvoyés est important.

Pour rechercher un document

  1. Tapez l'URL http://supportweb.remedy.com/bmc_cm.
  2. Dans le volet My Support Login, saisissez votre nom d'utilisateur et votre mot de passe, puis cliquez sur le bouton Login.
  3. Cliquez sur Documentation.
  4. Sélectionnez l'un des critères de recherche suivants :
  5. Un produit, par exemple : Policy Management.
  6. Une version, par exemple : 6.0.3.
    • Si vous sélectionnez un produit, vous pouvez sélectionner la version dans la liste d'éléments.
    • Si vous ne sélectionnez pas de produit, entrez le numéro de version dans le champ Version.
    • Pour consultez les versions antérieures ou ultérieures à une version, utilisez les symboles « > » (plus grand que) ou « < » (plus petit que). Par exemple, entrez « >6 » pour afficher les versions 6.0, 6.0.1, 6.0.2, 6.0.2.1, 6.0.3 et 6.5 disponibles.
    • Une langue, par exemple : français ou anglais.
    • Un type de document, par exemple : manuel.

      • Remarque : pour afficher la liste complète des documents BMC CM, ne sélectionnez aucun critère de recherche.

  7. Cliquez sur Rechercher Documentation.

    8. Les résultats de la recherche s'affichent.

    • Cliquez sur un en-tête de colonne des résultats de la recherche pour trier les documents selon cet en-tête de colonne.
    • Si les résultats affichent un nombre important de documents, cliquez sur le lien Next ou Previous en haut de la liste des résultats pour afficher les autres documents.
  8. Cliquez sur un document dans les résultats de la recherche pour le sélectionner.

    9. Une description de ce document s'affiche alors dans le champ Documentation Notes.

  9. Effectuez l'une des opérations suivantes :
    • Pour afficher le document sélectionné, cliquez sur View.
    • Vous ne pouvez afficher qu'un document à la fois. Pour afficher un autre document, vous devez fermer le document actuellement ouvert.
    • Pour enregistrer le document sélectionné sur le disque, cliquez sur Save To Disk.

      • Cliquez sur New Search pour effectuer une nouvelle recherche.

L'aide en ligne de Policy Manager est conçue pour être utilisée avec le Guide de l’administrateur de Policy Manager, également disponible sur le site Web de l’assistance à la clientèle BMC. Le Guide de l’administrateur de Policy Manager contient des informations sur la prise en charge des plates-formes, l'installation et les mises à niveau, le réglage du service d'annuaire (LDAP), la maintenance et le dépannage. Si vous souhaitez imprimer ce document, sachez qu'il inclut une table des matières, un index et que ses pages sont numérotées.

Les documents apparentés suivants sont également disponibles sur le site Web de l’assistance à la clientèle BMC :

    • Le Guide de configuration système requise de BMC Configuration Management répertorie les besoins en matériel (tels que vitesse de traitement, espace disque et mémoire vive) et la configuration minimale nécessaire pour les diverses plates-formes prises en charge. Ce document fournit également la liste des bases de données et des services d’annuaire pris en charge.
    • Le Guide de planification de BMC Configuration Management vous aide à concevoir une infrastructure BMC CM adaptée à votre entreprise.
    • Le Guide d'installation et de déploiement de BMC Configuration Management fournit des instructions pour une toute nouvelle installation (déploiement) de votre infrastructure BMC CM.
    • Le Guide de l'administrateur de Report Center (Report Center Administrator's Guide) fournit des instructions détaillées pour l'installation de tous les composants du module Inventory Management. Il fournit également la procédure d'exécution des requêtes portant sur les informations d'inventaire et de journalisation centralisée ainsi que sur la configuration des plugins d'inventaire (Inventory) et de journalisation (Logging).
    • Le Guide de l'administrateur de Server Management (Server Management Administrator’s Guide) est le principal document de référence pour les produits Server Management. Il fournit des informations conceptuelles sur l'utilisation de Server Management, y compris sur l'interface gérée par navigateur.
    • Le Guide de l'administrateur d'Application Packager (Application Packager Administrator’s Guide) explique de manière détaillée comment utiliser Application Packager pour mettre en package des logiciels afin de les distribuer sur des ordinateurs de bureau ou des serveurs. Il inclut également des informations sur l'utilisation de la ligne de commande, les policies, les modèles XML et les macros du système Windows.
    • Le Guide de référence de BMC Configuration Management fournit des informations de référence diverses sur les composants de l'infrastructure BMC CM, comme les options de ligne de commande, les propriétés de tuner, les propriétés, paramètres et états de channel et les codes de journalisation. La liste des propriétés de tuner et de package que vous pouvez définir dans Policy Manager figure dans ce guide. Il répertorie en outre les options de ligne de commande et les entrées de log pouvant être utiles pendant l’utilisation de Policy Manager.
    • Le Guide de mise à niveau de BMC Configuration Management (Upgrade Guide) fournit des instructions pour mettre à niveau votre infrastructure BMC CM à la dernière version.
    • D'autres documents susceptibles d’être utilisés dans certaines circonstances sont mentionnés dans les sections appropriées du présent document.
