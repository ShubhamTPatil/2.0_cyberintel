La première fois que vous cliquez sur le lien Aide dans une application, un avertissement de sécurité s'affiche. Il vous invite à installer et exécuter un applet doté d’une signature codifiée de Quadralay Corporation. Cet applet fournit la table des matières dynamique et la fonctionnalité de recherche dans les rubriques d'aide. Si vous n'installez pas l'applet, l'aide fonctionne quand même mais certaines fonctions ne sont pas disponibles.

L'aide est fournie avec l'application que vous utilisez et est mise à jour automatiquement lors de la mise à jour du channel du produit. Dans la mesure où l'aide fait partie d'un channel, les fichiers d'aide sont stockés sur votre poste de travail et donc toujours disponibles, même en l'absence d'une connexion réseau.

L'aide est contextuelle. Lorsque vous vous trouvez dans une page particulière de l'interface gérée par navigateur pour une application et que vous cliquez sur le lien Aide dans le coin supérieur droit de la page, l'aide qui s'affiche correspond à cette page de l'interface. Certaines rubriques comportent une liste de liens pointant vers d'autres rubriques d'aide (sous l'en-tête « Voir aussi »).

Pour lancer une recherche dans les rubriques, vous avez le choix entre les options suivantes :

    • Bouton Sommaire. Cliquez sur le bouton Sommaire de la fenêtre d'aide pour afficher le sommaire des rubriques d'aide.
    • Bouton Index. Cliquez sur le bouton Index de la fenêtre d'aide, puis passez en revue la liste alphabétique des entrées pour trouver le mot que vous cherchez. Cliquez sur l'entrée d'index souhaitée pour afficher la rubrique d'aide correspondante.
    • Bouton Rechercher. Cliquez sur le bouton Rechercher, entrez un ou plusieurs mots dans la zone de texte, puis appuyez sur la touche Entrée ou cliquez sur le bouton Valider. La page affiche la liste de toutes les rubriques de l'aide en ligne contenant le mot ou l'expression que vous avez entré. Si vous avez entré plusieurs mots, la fonction de recherche trouve les rubriques d’aide qui contiennent tous les mots que vous avez saisis. Les rubriques trouvées par la recherche sont classées par ordre de pertinence.

Revenir à la rubrique d'aide précédente. Si vous souhaitez revenir à une rubrique d'aide précédente, vous pouvez utiliser les raccourcis de navigation standard. Dans certains navigateurs, vous pouvez appuyer sur la touche Retour arrière du clavier pour revenir en arrière ou cliquer sur le bouton droit de votre souris et sélectionner Précédent dans le menu qui apparaît. La fenêtre d'aide ne comporte pas de bouton Précédent.
