Pour plus d'informations sur l'installation des composants mentionnés dans cette section, reportez-vous au Guide d'installation et de déploiement de BMC Configuration Management, disponible sur le site Web de l'assistance à la clientèle BMC. Pour plus d'informations sur la configuration de Patch Manager et de Patch Service, reportez-vous au Guide de l'administrateur de Patch Management (Patch Management Administrator's Guide). Pour obtenir des informations sur la configuration de Report Center et de Scanner Service, reportez-vous au Guide de l'administrateur de Report Center (Report Center Administrator’s Guide). Ces documents sont également disponibles sur le site Web de l'assistance à la clientèle BMC.

Côté serveur. Les composants suivants doivent être installés et configurés sur la même machine que Policy Manager :

    • Patch Manager
    • Report Center

Côté client. Les composants suivants doivent être installés sur les machines cibles avec Policy Service :

    • Patch Service
    • Scanner Service
