L'état d'attribution de groupe de patch indique si un groupe de patch doit être attribué à des machines cibles. Les deux états d'attribution de groupe de patch que vous pouvez définir dans Policy Manager sont les suivants :

    • Attribuer—Choisissez cet état si vous souhaitez attribuer un groupe de patch aux machines cibles spécifiées.
    • Exclure—Choisissez cet état si vous ne souhaitez pas attribuer un groupe de patch aux machines cibles spécifiées. Par exemple, si vous souhaitez attribuer un groupe de patch à un groupe de machines, à l'exception d'une machine, vous pouvez choisir l'état Attribuer pour l'ensemble du groupe et l'état Exclure pour la machine à exclure.

Il ne faut pas confondre l'état d'attribution de groupe de patch avec l'action de patch. L'action de patch désigne les différentes actions qui peuvent être exécutées sur chaque patch d'un groupe de patch, sur la machine cible, à savoir Installer, Mettre en attente, Désinstaller, ou Ne rien faire. Vous spécifiez l'action pour chaque patch du groupe de patch à l'aide de Patch Manager.
