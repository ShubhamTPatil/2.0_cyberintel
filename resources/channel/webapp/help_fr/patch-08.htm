Vous pouvez afficher une simulation d'installation de patch à condition d'avoir exécuté au moins une fois les channels de service suivants sur la machine cible :

    • Patch Service
    • Scanner Service

Lors de leur exécution, les channels de service collectent des informations sur l'installation simulée des patchs.

Lorsque vous simulez une installation de patch, la simulation s'effectue sur tous les groupes de patch qui s'appliquent à la cible sélectionnée et pas seulement sur ceux qui font partie de la policy. Si les groupes de patch ont été attribués à toutes les machines cibles, à un container, ou à un groupe dont la cible sélectionnée fait partie, alors ils devraient être inclus dans la simulation.

Remarque : pour simuler l'installation d'un patch, la cible que vous sélectionnez pour les groupes de patch doit être une seule machine.

Pour simuler l'installation de groupes de patch

Dans l'onglet Groupes de patch, cliquez sur Exécuter la simulation.

La page Simulation de l'installation du/des patchs s'affiche et indique l'ordre d'installation des patchs sur la machine cible. Cette page fournit également les informations suivantes :

    • Une brève description du patch
    • L'ID que Patch Manager a attribué au patch. Cet ID est différent de celui attribué par le vendeur du patch
    • L'action que vous avez spécifiée pour le patch à l'aide de Patch Manager
    • Les remarques du fournisseur relatives à ce patch

      • Pour afficher des informations plus détaillées sur ces patchs, cliquez sur Afficher les logs de simulation de patch. Pour plus d'informations, reportez-vous à la section Affichage des informations détaillées sur l'installation de groupes de patch.
