Pour afficher des informations détaillées sur l'installation de groupes de patch

Dans la page Simulation de l'installation du/des patch, cliquez sur Afficher logs détaillés.

La page Logs de simulation de patch s'affiche et fournit des informations détaillées sur les patchs à installer. Vous pouvez voir quels patchs sont obsolètes ou quels patchs ne peuvent être installés à cause de la présence de conflits.

Exemples d'entrées de log de simulation de patch :

Filtrer les groupes de patch

Processing stage MS04-004.Q832894.Q832894-6SP1.exe._1139234174

checking MS04-004.Q832894.Q832894-6SP1.exe._1139234174 for obsolescence

Processing install MS04-007.Q828028.Windows2000-KB828028-x86-ENU.exe._1592190521

checking MS04-007.Q828028.Windows2000-KB828028-x86-ENU.exe._1592190521 for obsolescence
