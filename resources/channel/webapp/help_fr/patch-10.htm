L'installation de certains patchs peut nécessiter le redémarrage des machines cibles. Vous pouvez utiliser Policy Manager pour indiquer le niveau de contrôle que vous souhaitez accorder aux utilisateurs, à l'aide des options disponibles dans l'onglet Paramètres de redémarrage, lorsque leurs machines doivent être redémarrées après réception d'un patch. Ces options définissent les propriétés qui contrôlent les différents aspects du redémarrage. Pour plus d'informations sur ces propriétés, reportez-vous à la section Avancé : propriétés de tuner pour les paramètres de redémarrage.

Cette procédure présuppose que vous vous trouvez à la page Modifier Policy. Pour plus d'informations, reportez-vous à la section Instructions générales pour la création et la modification de policies.

Pour gérer les paramètres de redémarrage pour les installations de patchs à l'aide de Policy Manager

  1. Cliquez sur l'onglet Paramètres de redémarrage > Patchs.

    2. L'onglet Paramètres de redémarrage s'affiche.

  2. Choisissez Utiliser Policy Manager pour gérer les paramètres de redémarrage de patchs.
  3. Choisissez une ou plusieurs des options suivantes :
    • Permet à la machine de redémarrer automatiquement après l'installation des patchs — Choisissez cette option si vous souhaitez que la machine cible redémarre automatiquement après l'installation d'un patch nécessitant le redémarrage du système.

      • Configurez les options suivantes pour avertir les utilisateurs que la machine va redémarrer :

    • Affiche des messages d'alerte à l'attention des utilisateurs avant le redémarrage de la machine  — Choisissez cette option si vous souhaitez que la machine cible affiche des messages d'alerte pour avertir les utilisateurs que la machine va redémarrer.
    • Ne pas autoriser les utilisateurs à reporter un redémarrage... — Choisissez cette option si vous souhaitez avertir les utilisateurs avant le redémarrage de la machine, sans toutefois les autoriser à le reporter. Le compte à rebours avant le redémarrage décrit dans l'option suivante se déclenche une fois l'installation des patchs effectuée.
    • Autoriser les utilisateurs à reporter un redémarrage pour une durée totale de _ minutes — Choisissez cette option si vous souhaitez autoriser les utilisateurs à reporter un redémarrage. Spécifiez le nombre de minutes à accorder aux utilisateurs pour le report du redémarrage. Une fois ce délai écoulé, l'utilisateur ne pourra plus différer le redémarrage et le compte à rebours du redémarrage commencera.
    • Compte à rebours avant le redémarrage — Indiquez la durée de l'attente ou du compte à rebours avant le redémarrage de la machine. Le compte à rebours avant le redémarrage laisse à l'utilisateur le temps d'enregistrer ses données avant le redémarrage de sa machine.

Lorsque vous avez terminé de modifier la policy, affichez l'aperçu, puis enregistrez-la. Pour plus d'informations, reportez-vous à la section Aperçu et enregistrement des modifications de policy.
