Installer Policy Manager et Report Center. La policy compliance nécessite la présence de Policy Manager et des channels de Report Center sur le serveur console. Si vous avez suivi les instructions pour installer et configurer le serveur console comme indiqué dans le Guide d'installation et de déploiement de BMC Configuration Manager, disponible sur le site Web de l'assistance à la clientèle BMC, vous devriez avoir les channels suivants installés et en bon fonctionnement :

    • Common Management Services
    • Policy Manager
    • Report Center

      • Important :  les informations de policy compliance sont basées sur le scan du composant tuner/transmitter. Si différentes planifications de scan sont définies pour les composants dans la page de configuration du plugin d'inventaire de Report Center et le composant tuner/transmitter n'est pas exécuté, sachez que Policy Manager peut ne pas afficher les informations de policy compliance correctes.

Assurez-vous que votre configuration système correspond à celle indiquée dans le guide Configuration système requise de BMC Configuration Management, disponible sur le site Web de l'assistance à la clientèle BMC.

Assurez-vous d'avoir installé et configuré Policy Manager comme indiqué dans le Guide de l'administrateur de Policy Management. Assurez-vous également d'avoir installé et configuré Report Center (sans oublier la configuration de la base de données et l'importation de la bibliothèque de requêtes) comme indiqué dans le Guide d'installatoin et de déploiement de BMC Configuration Management et le Guide de l'administrateur de Report Center. Tous ces documents sont disponibles sur le site Web de l'assistance à la clientèle BMC.

Activer la policy compliance. Par défaut, la collecte de données de policy compliance est désactivée depuis la version 6.0.2.1. Si vous souhaitez consulter les rapports de Policy Compliance, assurez-vous d'activer la collecte de données de Policy Compliance et de paramétrer les options de Policy Compliance pour répondre aux besoins de votre entreprise, comme décrit dans la section Configuration des paramètres de la fonction policy compliance.

Synchroniser le service d'annuaire et la base de données. Pour calculer la conformité, vous devez tout d'abord synchroniser le service d'annuaire avec la base de données à l'aide des paramètres système de la console (Console > Paramètres système > Source de données > Service de synchronisation LDAP/base de données). Pour plus d'informations, consultez l'aide en ligne des paramètres système de la console.
