Policy Manager vous oblige à afficher un aperçu des modifications de policy avant que vous puissiez les enregistrer. Vous êtes ainsi certain d'appliquer les paramètres qui conviennent à la cible. Lorsque vous enregistrez une policy, sachez que vous sauvegardez toutes les modifications qui ont été effectuées dans tous les onglets. Vérifiez bien toutes vos modifications dans tous les onglets avant de les appliquer.

La page Aperçu fonctionne en parallèle à la page Modification de policy. La page Aperçu contient les quatre onglets suivants qui correspondent chacun à un onglet de la page Modification des paramètres :

    • Packages—Indique les packages dans la policy, y compris des informations sur chaque package comme les états, les planifications et la priorité d'installation.
    • Période d'inactivité—Indique la période d'inactivité pour la cible.
    • Planification de policy—Indique la planification pour la mise à jour de Policy Service.
    • Avancé—Indique les paramètres avancés comme les propriétés du tuner et du package, les droits du transmitter et les informations de profil pour la cible.

Pour afficher un aperçu de vos modifications, les enregistrer, les annuler ou poursuivre les modifications

  1. Cliquez sur Aperçu après avoir modifié une policy.

    2. Lorsque vous cliquez sur Aperçu dans la page Modification de policy, la page Aperçu apparaît. Veillez à toujours cliquer sur chacun des onglets afin d'afficher les modifications apportées à ces sections.

  2. Si vous voulez enregistrer la policy, cliquez sur Enregistrer pour confirmer et enregistrer vos modifications.
  3. Si vous ne voulez pas enregistrer vos modifications et voulez continuer à modifier la policy, cliquez sur Retour à la modification.
  4. Si, à tout moment, vous voulez annuler vos modifications et retourner à la page Vue Cible, cliquez sur Annuler.
