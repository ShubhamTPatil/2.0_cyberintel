La page Vue Package offre une alternative pratique à la page Vue Cible. Vous pouvez l’utiliser lorsque vous voulez afficher les cibles auxquelles sont attribué(s) un ou plusieurs packages communs. Lorsque vous êtes dans la page Vue Package, vous pouvez ajouter des cibles, supprimer des cibles ou modifier des policies.

Pour ajouter ou supprimer des cibles sur la page Vue Package

  1. Cliquez sur l’onglet Vue Package.
  2. Sur le côté gauche de la page, sélectionnez un ou plusieurs packages. Pour plus d'informations, reportez-vous aux rubriques suivantes :
    • Exploration des packages
    • Recherche de packages
    • Affichage des cibles auxquelles un package a été attribué

      • Chaque fois que vous sélectionnez un package, celui-ci apparaît dans la liste des packages sélectionnés dans la partie droite de la page. Pour supprimer un package de cette liste, cliquez sur l'icône Supprimer située à côté.

      Sous la liste des packages sélectionnés se trouve une liste des cibles auxquelles tous les packages sélectionnés ont été directement attribués (l'intersection des policies des packages sélectionnés). Si une cible n'a pas reçu l'attribution de tous les packages sélectionnés, elle n'apparaît pas dans la liste. Les cibles qui s'affichent peuvent avoir déjà reçu les packages affichés.

  3. Voici comment ajouter des cibles :
    1. Cliquez sur Ajouter.
    2. Utilisez le côté gauche de la page pour y trouver une cible et la sélectionner. Pour plus d'informations, reportez-vous à la section Affichage des cibles.

      3. Chaque fois que vous sélectionnez une cible, celle-ci apparaît dans la liste des cibles sélectionnées dans la partie droite de la page.

    3. Vous pouvez également spécifier l'état du package et la planification pour chacune des cibles.
      • Pour l’état du package, choisissez un état sous la colonne État primaire ou État secondaire.
      • Pour la planification, sélectionnez les cibles auxquelles doit s'appliquer la planification et cliquez sur Définition d'une planification commune.

        • Pour plus d'informations, reportez-vous à la section Indication des états et des planifications pour les packages d'une policy.

    4. Quand vous avez terminé d'ajouter des cibles, cliquez sur Ajouter.
  4. Voici comment supprimer des cibles :
    1. Sélectionnez une ou plusieurs cibles desquelles vous voulez supprimer les packages communs.
    2. Cliquez sur Supprimer.
    3. Sur la page de confirmation, cliquez sur Supprimer.

Après avoir ajouté ou supprimé des cibles, vous revenez à la page Vue Package.

Pour modifier les policies à partir de la page Vue Package

  1. Cliquez sur l’onglet Vue Package.
  2. Sur le côté gauche de la page, sélectionnez un ou plusieurs packages. Pour plus d'informations, reportez-vous aux rubriques suivantes :
    • Exploration des packages
    • Recherche de packages
    • Affichage des cibles auxquelles un package a été attribué

      • Chaque fois que vous sélectionnez un package, celui-ci apparaît dans la liste des packages sélectionnés dans la partie droite de la page. Pour supprimer un package de cette liste, cliquez sur l'icône Supprimer située à côté.

      Sous la liste des packages sélectionnés se trouve une liste des cibles auxquelles tous les packages sélectionnés ont été directement attribués (l'intersection des policies des packages sélectionnés). Si une cible n'a pas reçu l'attribution de tous les packages sélectionnés, elle n'apparaît pas dans la liste. Les cibles qui s'affichent peuvent avoir déjà reçu les packages affichés.

  3. Cliquez sur Modifier.
  4. Procédez aux modifications de la policy. Pour plus d'informations, reportez-vous à la section Instructions générales pour la création et la modification de policies.

Après avoir modifié la policy, vous revenez à la page Vue Package.
