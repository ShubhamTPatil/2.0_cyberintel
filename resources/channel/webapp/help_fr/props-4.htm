Cette rubrique indique comment paramétrer les propriétés de package des packages. Les propriétés de package fournissent des informations sur le package et la façon dont il doit s'exécuter. Pour plus d'informations, reportez-vous à la section Définition d'une propriété de package.

Remarque : vous pouvez définir des paramètres au niveau de la cible, comme les propriétés de tuner et de package, pour une seule cible à la fois. En revanche, si vous êtes en mode de sélection multiple et que vous modifiez des policies pour plusieurs cibles à la fois, la définition de paramètres au niveau de la cible n'est pas autorisée. Vous pouvez toutefois définir des paramètres au niveau de la cible pour l'ensemble des cibles en sélectionnant la cible Toutes les machines cibles.

Cette procédure présuppose que vous vous trouvez dans l'onglet Packages de la page Modification de policy. Pour plus d'informations, reportez-vous à la section Instructions générales pour la création et la modification de policies.

Pour définir les propriétés de package pour une cible

  1. Dans la page Modification de policy, cliquez sur l'onglet Avancé > Propriétés de tuner/package.

    2. La page Propriétés de tuner et de package s'affiche.

  2. Pour ce qui concerne les propriétés de package, procédez comme suit dans le tableau Propriété de package :
    1. À partir de la liste Package, sélectionnez le package ou le groupe de packages dont vous souhaitez paramétrer les propriétés. Vous pouvez sélectionner l’une des options suivantes :
      • L'adresse URL d'un package spécifique — Sélectionnez cette option si vous voulez paramétrer la propriété d'un package spécifique.
      • Tous les packages — Sélectionnez cette option si vous voulez paramétrer la propriété pour tous les packages sur les tuners des machines cibles, à l'exception du channel Policy Service.
      • Tous les packages abonnés — Sélectionnez cette option si vous souhaitez paramétrer la propriété pour tous les packages qui font partie de la policy, à l'exception de ceux qui ont les états suivants : uninstall, exclude, primary ou advertise. De plus, le channel Policy Service n'est pas inclus.
      • Policy ServiceChannel — Sélectionnez cette option si vous voulez paramétrer la propriété du channel Policy Service uniquement.
    2. Entrez le nom et la valeur de la propriété.
    3. Cliquez sur Appliquer.

      4. La propriété est ajoutée à la zone de texte Propriétés.

      Remarque : vous pouvez également entrer directement l'adresse URL du package, le nom de la propriété de package et sa valeur dans la zone de texte Propriétés. Conformez-vous au format indiqué dans la section : Format des propriétés de tuner et de package.

  3. Aperçu des propriétés figurant dans la liste de la zone de texte Propriétés.

    4. Remarque : le fait de supprimer une propriété dans la zone de texte Propriétés ne suffit pas pour la supprimer des machines cibles. C'est pourquoi elle apparaît telle quelle sur le tuner cible (non géré par Policy Management). Pour obtenir des instructions sur la suppression des propriétés, reportez-vous à la section Suppression de propriétés de tuner et de package.

Lorsque vous avez terminé de modifier la policy, affichez l'aperçu, puis enregistrez-la. Pour plus d'informations, reportez-vous à la section Aperçu et enregistrement des modifications de policy.
