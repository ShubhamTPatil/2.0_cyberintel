La cible Toutes les machines cibles est composée de toutes les cibles (utilisateurs ou machines) sur lesquelles Policy Service est configuré sur le LDAP d'hébergement.

Vous pouvez utiliser la cible Toutes les machines cibles pour distribuer le logiciel à chaque personne de votre organisation. Elle permet aussi de paramétrer les propriétés de cible et de package de toutes les machines cibles afin de s'assurer que les utilisateurs puissent en avoir un usage cohérent.

L'icône Toutes les machines identifie la cible Toutes les machines cibles de votre page Vue cible. La cible Toutes les machines cibles est un groupe virtuel qui n'existe pas réellement dans le répertoire sous-jacent. Par conséquent, ce groupe ne nécessite pas d'être conservé avec une liste de toutes les machines cibles. Lorsqu'une modification survient (ajout ou suppression de machines cibles), la mise à jour de la cible Toutes les machines cibles s'effectue automatiquement.

Si vous souhaitez que toutes les machines cibles reçoivent un package, affectez-le à la cible Toutes les machines cibles. La cible Toutes les machines cibles permet aussi d'appliquer des propriétés de cible et de package à toutes les machines cibles afin de s'assurer que les utilisateurs puissent en avoir un usage cohérent.

Si vous souhaitez éviter que certaines cibles reçoivent des packages par ailleurs attribués à Toutes les machines cibles, vous pouvez les affecter séparément et leur définir un état Exclure. L'état Exclure remplace tous les autres états. Les packages comportant cet état ne sont pas distribués aux cibles qui leur sont attribuées.
