C'est en sélectionnant l'état Exclure pour une paire cible-package que vous identifiez une cible exclue. L'état Exclure définit que la cible ne reçoit pas le package associé, même s'il a été attribué par ailleurs, par exemple à un groupe cible. Cette fonction est utile quand vous voulez que tous les membres d'un large groupe cible, à l'exception de certains d'entre eux, tels que Toutes les machines cibles, reçoivent un package. Dans ce cas, vous pouvez affecter le package au groupe cible, créer des policies séparées aux cibles que vous souhaitez exclure et leur paramétrer l'état Exclure.
