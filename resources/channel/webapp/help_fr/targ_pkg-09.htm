Cette rubrique décrit comment afficher les contenus d'une cible qui peut être développée (comme par exemple les utilisateurs d'un groupe d'utilisateurs) à partir de la page Vue cible.

Pour afficher le contenu d'une cible qui peut être développée à partir de la page Vue cible

    • Sur la page Vue cible, cliquez sur le signe + situé à côté du nom de la cible qui peut être développée.

Conseil : si vous maintenez le curseur de votre souris sur les cibles, leurs noms distinctifs (DN) ou leurs unités organisationnelles (OU) dans le service d'annuaire s'affichent.

Lorsqu'une cible est développée pour afficher ses membres, vous pouvez voir les modifications suivantes sur le côté gauche de la page Vue cible :

    • Si vous développez un container, le titre figurant sur le côté droit de la page se modifie pour afficher le container. Le titre reste jusqu'à ce que vous naviguiez sur un autre container.
    • Si vous développez un groupe, les membres du groupe s'affichent et le nom du groupe est ajouté au fil d'Ariane pour vous orienter dans la hiérarchie de la cible.
    • Les options de recherche s'affichent à côté du haut de la partie droite de la page afin que vous puissiez rechercher une cible à l'intérieur du container concerné. Pour plus d'informations, reportez-vous à la section Recherche de cibles.
    • Un lien apparaît sous les options de recherche, pour vous permettre de passer du mode de sélection unique au mode de sélection multiple.
    • Un fil d'Ariane s'affiche sous le lien du mode de sélection pour vous montrer votre emplacement dans le container et la hiérarchie de groupe. Ceci vous permet de naviguer aisément dans la hiérarchie. Le fil d'Ariane vous montre votre position actuelle dans la hiérarchie de container. Les éléments soulignés sont les liens qui vous permettent de revenir à une autre position au sein de la hiérarchie. L'élément suivant le lien de la page d'accueil affiche les containers de niveau supérieur, c'est-à-dire, généralement, le domaine racine.
    • Si la liste développée contient plus d'éléments que le nombre prédéfini, le lien suivant devient actif pour vous permettre d'accéder aux autres éléments de la liste. Utilisez les liens suivant et précédent pour naviguer en avant et en arrière dans de longues listes. Vous pouvez utiliser la liste déroulante pour accélérer votre navigation dans de très longues listes.

      • Remarque : Active Directory et ADAM n'obtiennent pas le nombre total d'entrées tant que vous n'avez pas fait défiler l'ensemble des résultats de la recherche. Par conséquent, la liste déroulante affiche « ... » pour le nombre total d'entrées, jusqu'à ce que vous ayez atteint la dernière page.

    • Si un groupe contient un quelconque membre invalide, celui-ci s'affiche avec un point d'interrogation et son DN apparaît grisé et en italiques. Si vous maintenez le curseur de votre souris sur le point d'interrogation, les informations relatives au membre invalide s'affichent, comme présenté dans la Figure 3-2.
