Le côté droit de la page Vue cible affiche les packages attribués à la ou les cibles sélectionnée(s) dans la liste cible. L'affichage peut quelque peu varier selon le mode sélectionné. Pour passer d'un mode à l'autre, cliquez sur le lien Unitaire ou Multiple situé en haut de la liste Cibles. Ces deux modes sont traités dans les sections suivantes :

    • Mode de sélection unitaire
    • Mode de sélection multiple

Mode de sélection unitaire

En mode de sélection unitaire, vous pouvez choisir une cible unique et afficher les packages lui ayant été affectés directement ou indirectement.

Pour afficher les packages attribués à une cible unique

  1. Sur le côté gauche de la page Vue cible, assurez-vous que vous êtes en mode de sélection unitaire. Si vous êtes en mode de sélection multiple, cliquez sur le lien Unitaire pour passer en mode de sélection unitaire.
  2. Dans la partie gauche de la page Vue cible, localisez et sélectionnez la cible dont vous souhaitez afficher les packages :

    3. Après avoir cliqué sur une cible, tous les packages déjà attribués s'affichent dans la partie droite de la page. La cible sélectionnée peut avoir déjà reçu les packages affichés.

    Remarque : la résolution des conflits d'état n'est pas effectuée, il est donc possible que deux packages portant le même nom (ou adresse URL) mais ayant des états différents s'affichent.

Le nom de la cible sélectionnée s'affiche en haut du côté droit de la page. (Il s'agit généralement de l'attribut cn de l'entrée du service d'annuaire.) Sous le nom de la cible se trouve un tableau qui présente la liste des packages assignés (directement et indirectement) à la cible sélectionnée.

En mode de sélection unitaire, vous disposez des options suivantes :

Vue détaillée — Cliquez pour afficher les états et les planifications de chaque package attribué à la cible que vous avez sélectionnée.

Modifier — Cliquez pour créer ou modifier la policy de la cible. Pour plus d'informations, reportez-vous à la section Création et modification des policies.

Remarque : lorsque vous modifiez une policy, vous ne voyez que les packages qui sont directement attribués à une cible. Pour modifier les policies comprenant des packages attribués indirectement, vous devez sélectionner les cibles auxquelles ils sont directement attribués. Pour savoir à quelle cible les packages sont directement attribués, regardez dans la colonne Directement attribué à. Pour plus d'informations sur le concept d'attributions directes, reportez-vous à la section Définition d'une cible en attribution directe

Les colonnes suivantes s'affichent dans le tableau situé sur le côté droit de la page :

Packages — Indique les noms des packages attribués à la cible sélectionnée. Placez le curseur de votre souris sur le nom du package pour voir son adresse URL.

États primaire et secondaire — Indique l'état ou les états qui définissent la façon dont le package doit être distribué à chaque machine cible. Pour consulter la liste des états d'installation, reportez-vous à la section Présentation des états d'installation.

Directement attribué à — Indique la cible à laquelle le package a été directement attribué. Le package apparaît dans la liste du fait que la cible sélectionnée est membre d'un groupe (ou la cible elle-même) auquel le package a été directement attribué.

Mode de sélection multiple

En mode de sélection multiple, vous pouvez choisir deux cibles ou plus et consulter les packages qu'elles ont en commun.

Pour afficher les packages attribués à plusieurs cibles

  1. Sur le côté gauche de la page Vue cible, si vous n'êtes pas encore en mode de sélection multiple, vous pouvez y passer en cliquant sur le lien Multiple.
  2. Dans la partie gauche de la page Vue cible, trouvez et sélectionnez les cibles dont vous souhaitez afficher les packages.

    3. Chaque fois que vous sélectionnez une cible, celle-ci apparaît dans la liste des cibles sélectionnées dans la partie droite de la page. Pour supprimer une cible de cette liste, cliquez sur l'icône Supprimer située à côté.

    Sous la liste des cibles sélectionnées se trouve une liste des packages directement attribués à toutes les cibles sélectionnées (l'intersection des policies des cibles sélectionnées). Si un package n'est pas attribué à toutes les cibles sélectionnées, il n'apparaît pas dans la liste. La cible sélectionnée peut avoir déjà reçu les packages affichés.

En mode de sélection multiple, vous disposez des options suivantes :

Vue détaillée — Cliquez pour afficher les états et les planifications de chaque package attribué aux cibles que vous avez sélectionnées.

Modifier — Cliquez pour créer ou modifier la policy des cibles. Pour plus d'informations, reportez-vous à la section Création et modification des policies. Lorsque vous modifiez les policies de plusieurs cibles, vous ne pouvez ajouter, modifier et supprimer que les packages qui sont attribués directement à toutes les cibles.

Les colonnes suivantes s'affichent dans le tableau situé sur le côté droit de la page :

Packages — Indique les noms des packages attribués aux cibles sélectionnées. Placez le curseur de votre souris sur le nom du package pour voir son adresse URL.

États primaire et secondaire — Indique l'état ou les états qui définissent la façon dont le package doit être distribué à chaque machine cible. Pour consulter la liste des états d'installation, reportez-vous à la section Présentation des états d'installation.
