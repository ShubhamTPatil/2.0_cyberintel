Lorsque vous utilisez Policy Manager, vous sélectionnez des packages et les affectez aux cibles. Un package est généralement une application que vous souhaitez installer sur les machines cibles, mais il peut être composé de n'importe quelle donnée que vous souhaitez distribuer. En général, vous pouvez utiliser BMC CM Application Packager pour créer un package.

Une fois que vous attribuez un package à une cible, vous pouvez affecter un état à chaque paire package-cible, ainsi que des planifications et des paramètres de propriété affectant l'installation de chaque machine cible.

L'icône de package identifie les packages.
