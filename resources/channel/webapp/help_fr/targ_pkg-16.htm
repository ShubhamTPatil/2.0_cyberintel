Vous pouvez rechercher les packages attribués à une ou plusieurs cibles. Cette option est utile lorsque vous utilisez un grand nombre de packages.

Pour rechercher un package

  1. Sur le côté gauche de la page Vue package, saisissez le nom d'un package dans la zone de texte Rechercher.

    2. Vous n'avez pas besoin de connaître le nom complet du package. Vous pouvez utiliser l'astérisque (*) en tant que caractère de remplacement pour zéro ou plusieurs caractères du nom. Si un astérisque est présent dans le nom, vous devez le faire précéder d'un caractère d'échappement pour qu'il ne soit pas considéré comme un caractère générique. Pour plus d'informations, reportez-vous à la section Caractères spéciaux dans les chaînes de recherche.

    L'opération de recherche ne fait aucune distinction entre les majuscules et les minuscules.

  2. Cliquez sur Valider ou appuyez sur la touche Entrée de votre clavier.

Les résultats de la recherche sont affichés dans la liste des packages. Cliquez sur le lien Réinitialiser pour afficher l'ensemble des packages attribués.

Si la liste contient plus d'éléments que le nombre prédéfini, le lien suivant devient actif pour vous permettre d'accéder aux autres éléments de la liste. Utilisez les liens suivant et précédent pour naviguer en avant et en arrière dans de longues listes. Vous pouvez utiliser la liste déroulante pour accélérer votre navigation dans de très longues listes.

Remarque : Active Directory et ADAM n'obtiennent pas le nombre total d'entrées tant que vous n'avez pas fait défiler l'ensemble des résultats de la recherche. Par conséquent, la liste déroulante affiche « ... » pour le nombre total d'entrées, jusqu'à ce que vous ayez atteint la dernière page.
