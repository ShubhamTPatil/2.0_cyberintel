Les cibles auxquelles ont été attribués les packages que vous avez sélectionnés dans la liste des packages s'affiche sur le côté droit de la page Vue package. L'affichage peut quelque peu varier selon le mode sélectionné. Pour passer d'un mode à l'autre, cliquez sur le lien Unitaire ou Multiple situé en haut de la liste des packages. Ces deux modes sont traités dans les sections suivantes :

    • Mode de sélection unitaire
    • Mode de sélection multiple

Mode de sélection unitaire

En mode de sélection unitaire, vous pouvez choisir un package unique et afficher les cibles auxquelles il a été affecté.

Pour afficher les cibles auxquelles un package unique a été attribué

  1. Sur le côté gauche de la page Vue package, assurez-vous que vous êtes en mode de sélection unitaire. Si vous êtes en mode de sélection multiple, cliquez sur le lien Unitaire pour passer en mode de sélection unitaire.
  2. Dans la partie gauche de la page Vue package, trouvez et sélectionnez le package dont vous souhaitez afficher les cibles.

    3. Après avoir cliqué sur un package, toutes les cibles auxquelles il a déjà été attribué s'affichent dans la partie droite de la page. Les cibles qui s'affichent peuvent avoir déjà reçu le package affiché.

Le nom du package sélectionné s'affiche en haut du côté droit de la page. Sous le nom du package se trouve un tableau qui présente la liste des cibles auxquelles le package sélectionné a été attribué.

En mode de sélection unitaire, vous disposez des options suivantes :

Vue détaillée — Cliquez pour afficher les états et les planifications de chaque package attribué à la cible que vous avez sélectionnée.

Modifier — Cliquez après avoir coché les cases correspondant aux cibles pour lesquelles vous souhaitez définir des états et des planifications. Pour plus d'informations, reportez-vous à la section Création et modification des policies.

Les colonnes suivantes s'affichent dans le tableau situé sur le côté droit de la page :

Cibles — Indique les noms des cibles auxquelles le package sélectionné a été attribué. (Il s'agit généralement de l'attribut cn de l'entrée du service d'annuaire.)

Conseil : si vous maintenez le curseur de votre souris sur une cible, son unité organisationnelle dans le service d'annuaire s'affiche.

États primaire et secondaire — Indique l'état ou les états qui définissent la façon dont le package doit être distribué à chaque machine cible. Pour consulter la liste des états d'installation, reportez-vous à la section Présentation des états d'installation.

Mode de sélection multiple

En mode de sélection multiple, vous pouvez choisir deux packages ou plus et consulter les cibles auxquelles ils ont été attribués.

Pour afficher les cibles auxquelles plusieurs packages ont été attribués

  1. Sur le côté gauche de la page Vue package, si vous n'êtes pas encore en mode de sélection multiple, vous pouvez y passer en cliquant sur le lien Multiple.
  2. Dans la partie gauche de la page Vue package, trouvez et sélectionnez les packages dont vous souhaitez afficher les cibles.

    3. Chaque fois que vous sélectionnez un package, celui-ci apparaît dans la liste des packages sélectionnés dans la partie droite de la page. Pour supprimer un package de cette liste, cliquez sur l'icône Supprimer située à côté.

    Sous la liste des packages sélectionnés se trouve une liste des cibles auxquelles tous les packages sélectionnés ont été directement attribués (l'intersection des policies des packages sélectionnés). Si une cible n'a pas reçu l'attribution de tous les packages sélectionnés, elle n'apparaît pas dans la liste. Les cibles qui s'affichent peuvent avoir déjà reçu les packages affichés.

En mode de sélection multiple, vous disposez des options suivantes :

Vue détaillée — Cliquez pour afficher les états et les planifications de chaque package attribué à la cible que vous avez sélectionnée.

Modifier — Cliquez après avoir coché les cases correspondant aux cibles pour lesquelles vous souhaitez définir des états et des planifications. Sur la page Modification de policy, vous pouvez paramétrer les états et les planifications des cibles que vous avez sélectionnées.

Les colonnes suivantes s'affichent sur le côté droit de la page :

Cibles — Indique les noms des cibles auxquelles le package sélectionné a été attribué. (Il s'agit généralement de l'attribut cn de l'entrée du service d'annuaire.)

Conseil : si vous maintenez le curseur de votre souris sur une cible, son unité organisationnelle dans le service d'annuaire s'affiche.

États primaire et secondaire — Indique l'état ou les états qui définissent la façon dont le package doit être distribué à chaque machine cible. Pour consulter la liste des états d'installation, reportez-vous à la section Présentation des états d'installation.
