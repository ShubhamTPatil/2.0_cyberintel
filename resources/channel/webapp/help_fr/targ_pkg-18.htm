La page Détails du package présente toutes les cibles auxquelles le ou les packages sélectionnés ont été attribués, ainsi que la priorité d'installation, les états primaire et secondaire et les planifications de chaque cible.

Pour afficher les détails d'un package (ou de plusieurs packages si vous êtes en mode de sélection multiple)

  1. Sur la page Vue package, sélectionnez un ou plusieurs packages si vous êtes en mode de sélection multiple.
  2. Cliquez sur le lien Vue détaillée pour afficher les détails du package.

    3. La page Détails du package s'affiche. Cette page présente les cibles auxquelles le ou les packages ont été attribués, ainsi que la priorité d'installation, les états primaire et secondaire et les planifications de chaque cible.

    Vous pouvez visualiser les détails des packages sélectionnés ou modifiez leur attribution aux cibles. Cliquez sur le nom de la colonne pour effectuer un tri par contenu. Faites défiler les longues listes en cliquant sur les liens précédent et suivant.

  3. Pour revenir à la page Vue package, cliquez sur le lien Vue basique.

Tri de la liste des cibles

Sur la page Détails du package, vous pouvez trier la liste des cibles selon :

    • Les planifications
    • Le nom de la cible

Pour effectuer le tri de la liste des cibles

Sur la page Détails de package, cliquez sur le nom de la colonne pour trier les cibles à l'aide de cette colonne (par ordre croisant ou décroissant).

    • Si vous effectuez votre tri en fonction des colonnes de planification, les cibles sont organisées selon la date et l'heure d'activation de cette planification. Par exemple, si vous effectuez votre tri en fonction de la planification primaire, les cibles sont organisées selon la date et l'heure paramétrées pour la planification primaire.
    • Si vous effectuez le tri en fonction du nom de la cible, les cibles sont organisées par ordre alphabétique.
